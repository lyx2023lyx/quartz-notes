> **章节目标**：理解梅特波利斯光照传输(MLT)的基本原理，掌握基于马尔可夫链的采样方法，了解路径采样策略的改进。

## 7.1 基本算法及其原理

### 7.1.1 梅特波利斯算法概述

**核心思想**：梅特波利斯算法是一种使采样概率密度函数**正比于目标函数**的采样方法，它通过产生局部相关的建议样本（突变），然后通过一定的取舍概率来选择或拒绝建议样本。

**与传统蒙特卡洛的区别**：
- 传统蒙特卡洛：使用全局相互独立的样本
- 梅特波利斯算法：产生局部相关的建议样本

**应用场景**：
- 复杂可见性分布的场景（大量随机方向被遮挡）
- 具有大量光泽面的场景（直接连通光源和摄像机的路径很少）
- 需要找到"困难路径"的场景

### 7.1.2 基本算法流程

```
算法：基本梅特波利斯光照传输
─────────────────────────────
输入：初始路径 x₀，目标分布函数 f(x)
输出：渲染图像

1. 初始化：
   - 从双向路径追踪中获取初始路径 x₀
   - 计算初始贡献值 f(x₀)

2. 迭代过程（对每个像素）：
   for i = 0 to N-1:
       a. 生成突变路径：
          x' ← T(xᵢ → x')  // 使用突变策略
       
       b. 计算接受概率：
          a(xᵢ, x') = min(1, [f(x')T(x'→xᵢ)]/[f(xᵢ)T(xᵢ→x')])
       
       c. 接受/拒绝决策：
          if random() < a(xᵢ, x'):
              xᵢ₊₁ ← x'      // 接受突变
          else:
              xᵢ₊₁ ← xᵢ      // 拒绝突变，保持原路径
       
       d. 累积样本到图像：
          image[pixel(xᵢ₊₁)] += contribution(xᵢ₊₁)

3. 归一化图像
```

### 7.1.3 数学原理

**细致平衡条件（Detailed Balance）**：

梅特波利斯算法满足细致平衡条件，保证马尔可夫链收敛到目标分布：

$$
f(x)T(x \to x')a(x, x') = f(x')T(x' \to x)a(x', x)
$$

**接受概率推导**：

设建议样本的接受概率为 $a(x, x')$，根据细致平衡条件：

$$
a(x, x') = \min\left(1, \frac{f(x')T(x' \to x)}{f(x)T(x \to x')}\right)
$$

**白话解读**：
- $f(x)$：路径 $x$ 的贡献值（测量贡献函数）
- $T(x \to x')$：从当前路径 $x$ 转移到建议路径 $x'$ 的转移概率
- 当 $f(x') > f(x)$ 时，新路径更可能被接受
- 当转移概率不对称时，需要补偿

**归一化常数问题**：

MLT算法中，我们只需要知道 $f(x)$ 的相对比例，不需要知道其绝对值。设真实的目标函数为 $f^*(x) = f(x)/b$，其中 $b = \int f(x)dx$ 是归一化常数。

在计算接受概率时：

$$
\frac{f^*(x')}{f^*(x)} = \frac{f(x')/b}{f(x)/b} = \frac{f(x')}{f(x)}
$$

**归一化常数 $b$ 会被约去**，因此MLT算法不需要预先知道 $b$ 的值。

### 7.1.4 初始路径生成

```
初始路径生成算法
────────────────
1. 使用双向路径追踪(BPT)生成一组候选路径
2. 根据路径的贡献值 f(x) 进行重要性采样
3. 选择贡献值较大的路径作为初始路径 x₀

优势：
- 避免从低贡献区域开始
- 加快收敛速度
- 确保初始路径是"有效"的
```

### 7.1.5 图像估计

**基本估计公式**：

对于像素 $j$ 的值 $I_j$：

$$
I_j = \frac{1}{N} \sum_{i=1}^{N} \frac{f_j(x_i)}{p(x_i)}
$$

其中：
- $f_j(x_i)$：路径 $x_i$ 对像素 $j$ 的贡献
- $p(x_i)$：路径 $x_i$ 的概率密度
- $N$：总样本数

**MLT的特殊处理**：

在MLT中，样本是相关的，因此估计公式调整为：

$$
I_j = \frac{b}{N} \sum_{i=1}^{N} \delta_j(x_i)
$$

其中：
- $b$：归一化常数（从BPT预计算）
- $\delta_j(x_i)$：指示函数，当路径 $x_i$ 贡献给像素 $j$ 时为1

## 7.2 原采样空间的突变策略

### 7.2.1 原采样空间（Primary Sample Space）

**定义**：原采样空间是所有随机数 $u \in [0,1]^d$ 构成的 $d$ 维单位超立方体空间。

**路径映射函数**：

$$X = P^{-1}(u)$$

其中：
- $u$：原采样空间的随机数向量
- $X$：路径空间的路径
- $P^{-1}$：从随机数到路径的映射函数（通过光线追踪实现）

**优势**：
- 路径空间维度不固定，原采样空间维度固定
- 在原采样空间进行突变更简单
- 可以使用传统的MCMC突变策略

### 7.2.2 PSSMLT算法

**基本思想**：直接在原采样空间进行梅特波利斯随机行走。

**算法流程**：

```
算法：PSSMLT (Primary Sample Space MLT)
────────────────────────────────────────
输入：原采样空间维度 d，迭代次数 N
输出：渲染图像

1. 初始化：
   u₀ ← 从[0,1]^d均匀采样
   x₀ ← P⁻¹(u₀)
   f₀ ← f(x₀)

2. 迭代过程：
   for i = 0 to N-1:
       a. 生成突变：
          u' ← uᵢ + ε·N(0,1)  // 高斯突变
          // 处理边界：
          if u' < 0 or u' > 1:
              u' ← u' mod 1   // 环绕处理
       
       b. 映射到路径空间：
          x' ← P⁻¹(u')
          f' ← f(x')
       
       c. 计算接受率：
          a ← min(1, f'/fᵢ)
       
       d. 接受/拒绝：
          if random() < a:
              uᵢ₊₁ ← u', fᵢ₊₁ ← f'
          else:
              uᵢ₊₁ ← uᵢ, fᵢ₊₁ ← fᵢ
       
       e. 累积到图像：
          image[pixel(xᵢ₊₁)] += contribution

3. 归一化
```

**突变策略**：

1. **小步幅突变（Small Step）**：
   $$
   u' = u + \epsilon \cdot \mathcal{N}(0, 1)
   $$
   - 用于探索局部区域
   - 保持样本相关性

2. **大步幅突变（Large Step）**：
   $$
   u' \sim \text{Uniform}([0,1]^d)
   $$
   - 用于全局探索
   - 避免陷入局部最优

3. **混合策略**：
   - 小步幅：概率 $p_{small}$
   - 大步幅：概率 $p_{large} = 1 - p_{small}$
   - 通常 $p_{small} \approx 0.7$

### 7.2.3 MMLT算法

**问题**：PSSMLT中，某些区域由于几何配置原因，很难找到有效路径。

**MMLT（Multiplexed MLT）解决方案**：引入多个采样技术（sampling techniques）。

**复合原采样空间**：

$$C^k = \{0, 1, \cdots, k+1\} \times U^k$$

其中：
- 第一个维度：采样技术索引 $t \in \{0, 1, \cdots, k+1\}$
- 第二个维度：原采样空间 $U^k = [0,1]^k$

**目标函数**：

$$C_t(u) = \frac{w_t(X) f(X)}{p_t(X)}$$

其中：
- $w_t(X)$：复合重要性采样权重
- $p_t(X)$：第 $t$ 个采样技术的概率密度

**算法流程**：

```
算法：MMLT (Multiplexed MLT)
───────────────────────────
1. 对采样技术进行突变：
   - 以一定概率改变当前采样技术 t
   - 保持随机数 u 不变

2. 对随机数进行突变：
   - 在当前采样技术 t 下
   - 使用PSSMLT的突变策略

3. 组合突变：
   - 同时改变采样技术和随机数
```

**序列回火（Serial Tempering）**：

允许马尔可夫链在不同采样技术之间跳跃，避免被"卡住"。

**权重系数**：

$$w_t(X) = \frac{p_t(X)}{\sum_{i} p_i(X)}$$

这是平衡启发式（balance heuristic）的权重计算。

### 7.2.4 原采样空间的优缺点

**优点**：
- 实现简单
- 突变策略直观
- 方差较低（目标分布相对平坦）

**缺点**：
- 缺乏对路径几何的感知
- 可能导致涟漪效应（ripple effect）
- 难以处理复杂的几何约束

## 7.3 路径空间的突变策略

### 7.3.1 流形探索（Manifold Exploration）

**核心思想**：利用费马原理和微分几何知识，探索镜面/光泽路径的局部邻域。

**理论基础 - 费马原理**：

光永远选择使传播时间最短的路径。这意味着：
- 反射定律：入射角等于反射角
- 折射定律：斯涅尔定律

**镜面流形（Specular Manifold）**：

对于包含镜面顶点的路径，所有镜面顶点满足反射/折射约束，形成一个低维流形。

**约束函数**：

对于镜面顶点 $x_i$，约束函数为：

$$
c_i(x_{i-1}, x_i, x_{i+1}) = T(x_i)^T h(x_i, \vec{x_{i-1}x_i}, \vec{x_ix_{i+1}})
$$

其中：
- $T(x_i)$：顶点 $x_i$ 处的正切空间基向量
- $h$：泛化半矢量

**流形定义**：

$$S = \{x \mid C(x) = 0\}$$

其中 $C: \mathbb{R}^{2n} \to \mathbb{R}^{2p}$ 是所有镜面顶点约束的集合。

**雅可比矩阵计算**：

$$J = \frac{\partial C}{\partial X} = \begin{bmatrix} B_1 & C_1 & & \\ A_2 & B_2 & C_2 & \\ & \ddots & \ddots & \ddots \\ & & A_{n-1} & B_{n-1} \end{bmatrix}$$

**正切空间**：

$$TS(x) = -A^{-1}[B_1, B_k]$$

**流形行走算法**：

```
算法：流形上的行走 (WalkManifold)
────────────────────────────────
输入：当前路径 x₁,...,xₙ，目标终点 xₙ'
输出：突变路径

1. 初始化：
   i ← 0, β ← 1

2. 迭代直到收敛：
   while ||xₙ - xₙ'|| > εL and i < N:
       
       a. 计算试探顶点：
          p ← x₂ - β·T(x₂)·P₂·A⁻¹·Bₖ·T(xₙ)ᵀ·(xₙ - xₙ')
       
       b. 光线追踪投影：
          沿方向 x₁→p 进行光线追踪
          通过所有镜面反射/折射
          得到预测路径 x₂⁺,...,xₙ⁺
       
       c. 判断是否接受：
          if ||xₙ⁺ - xₙ'|| < ||xₙ - xₙ'||:
              更新当前路径
              β ← min(1, 2β)  // 增大步长
          else:
              β ← 0.5β        // 减小步长
       
       d. i ← i + 1

3. 返回突变路径
```

**白话解读**：
- $\beta$：自适应步长参数
- $\epsilon$：收敛阈值
- $L$：场景最大尺寸
- $N$：最大迭代次数

**牛顿迭代法的应用**：

流形行走本质上是牛顿迭代法的变体：

$$
x_{n+1} = x_n - \frac{f(x_n)}{f'(x_n)}
$$

在我们的场景中：
- 目标：使终点逼近 $x_n'$
- 工具：使用正切空间的一阶泰勒近似
- 修正：通过光线追踪将近似路径投影回流形

**接受概率**：

$$
a(x, x') = \min\left(1, \frac{f(x')T(x' \to x)}{f(x)T(x \to x')}\right)
$$

其中转移概率需要考虑：
- 方向采样的概率
- 泛化几何项
- 雅可比行列式

### 7.3.2 半矢量空间光照传输

**核心思想**：使用半矢量（half-vector）作为约束，统一处理镜面和光泽表面。

**路径空间转换**：

将路径从顶点空间转换到半矢量空间：

$$H^\perp = (x_0, h_1^\perp, h_2^\perp, \cdots, h_{k-1}^\perp, x_k) \in \Omega(h^\perp)$$

**半矢量定义**：

$$
\vec{h} = \frac{\eta(x,v)v + \eta(x,w)w}{||\eta(x,v)v + \eta(x,w)w||}
$$

其中：
- $v$：入射方向
- $w$：出射方向
- $\eta$：折射率

**雅可比行列式**：

半矢量空间到路径空间的转换：

$$
\frac{dX}{dH^\perp} = G(x_0 \leftrightarrow \cdots \leftrightarrow x_k) \cdot \prod_{i=1}^{k-1} \frac{(\langle h,i \rangle + \eta\langle h,o \rangle)^2}{\eta^2 |\langle h,o \rangle|}
$$

**BSDF突变**：

1. 固定路径两端 $x_0$ 和 $x_k$
2. 对所有半矢量同时采样得到增量
3. 使用牛顿法计算满足半矢量增量的合法路径

**最优步幅计算**：

对于Beckmann分布：

$$
s_{||max} = \frac{a}{\sqrt{\pi}}
$$

其中 $a$ 是粗糙度参数。

**图像平面突变**：

1. 对像素顶点执行像素大小的偏移
2. 计算摄像机光线第一个交点的微分
3. 通过雅可比矩阵转换为半矢量增量

### 7.3.3 综合突变策略

**分离策略（Separation Strategy）**：

将路径分成三段：

```
路径分段示意图
────────────────

xk ──...── xb ──...── xc ──...── x₁ ── x₀
│           │         │
│           │         └── 固定子路径
│           └── 半矢量空间子路径
└── 多链子路径

各段处理：
• 多链子路径 (xk,...,xb)：图像平面突变
• 半矢量空间子路径 (xb₋₁,...,xc₊₁)：BSDF突变  
• 固定子路径 (xc,...,x₁,x₀)：保持不变
```

**选择策略**：

$$
\begin{aligned}
P(x_i = x_b) &\propto a_i \\
P(x_0 = x_b) &= P(x_k = x_b) \approx 0.1 \\
P(x_i = x_c) &\propto a_i \cdot ||x_i - x_{i+1}||^2
\end{aligned}
$$

其中 $a_i$ 是顶点 $x_i$ 处的粗糙度。

## 7.4 反向路径采样在MLT算法中的运用

### 7.4.1 问题分析

**MMLT的问题**：

当突变采样技术时，原采样空间的位置 $u$ 保持不变，但由于不同采样技术使用不同的映射 $P_t^{-1}$，路径可能发生巨大变化：

$$X = P_t^{-1}(u) \neq P_{t'}^{-1}(u)$$

这导致采样技术突变的接受率极低。

**解决方案**：

在切换采样技术时，同时调整原采样空间的位置 $u$，使得路径保持不变：

$$P_t^{-1}(u) = P_{t'}^{-1}(v)$$

### 7.4.2 反向路径采样

**可逆跳跃马尔可夫链蒙特卡洛（RJMCMC）**：

[Bitterli et al., 2017] 提出RJMLT算法，实现采样技术之间的确定性跳跃。

**核心映射**：

$$h_{ij}(u) = (P_j^{-1})^{-1}(P_i^{-1}(u))$$

其中 $(P_j^{-1})^{-1}$ 是反向采样技术，将路径映射为随机数。

**雅可比行列式**：

$$|J[h_{ij}](u)| = \frac{|J[P_i^{-1}](u)|}{|J[P_j^{-1}](v)|} = \frac{p_j(v)}{p_i(u)}$$

**接受概率**：

$$
r(u \to v) = \frac{T(j \to i) w_j(v)}{T(i \to j) w_i(u)}
$$

**简化**：

设置 $T(i \to j) = w_j(u)$，则接受率变为 1。

**扩展路径空间**：

对于不可逆采样技术，引入辅助维度 $\gamma \in [0,1]^m$：

$$S^{-1}(x, \gamma)$$

使用概率可逆（probabilistic inverse）：

$$u_1 = a + \gamma \cdot (b - a)$$

### 7.4.3 RJMLT算法

**三种突变策略**：

1. **大步幅突变**：概率 10%
2. **MMLT小步幅突变**：概率 85%
3. **采样技术间突变**：概率 5%

**优势**：
- 采样技术突变接受率高
- 不需要重新执行路径追踪
- 可以动态切换到最优采样技术

### 7.4.4 原采样空间与路径空间的组合

**双向通道**：

```
原采样空间 ⟷ 路径空间
──────────────────────
正向：X = P⁻¹(u)
反向：u = P(X)
```

**组合算法**[Otsu et al., 2017]**：**

```
算法：空间组合突变
──────────────────
1. 在原采样空间记录当前状态 u
2. 随机选择突变空间：
   - 原采样空间：直接执行PSSMLT/MMLT突变
   - 路径空间：
     a. 正向映射：X = P⁻¹(u)
     b. 执行MLT/流形探索突变：X → Y
     c. 反向映射：v = P(Y)
3. 在原采样空间计算接受率
4. 接受/拒绝突变
```

**采样技术选择**：

对于路径空间的突变，保持当前采样技术不变，因为路径突变本身就能覆盖整个路径空间。

## 7.5 组合MC与MCMC算法

### 7.5.1 平行回火（Parallel Tempering）

**核心思想**：使用多个平行的马尔可夫链，每个链对应不同的"温度"（目标函数抽象程度）。

**温度参数**：

- 低温链：接近原始目标函数，探索局部
- 高温链：更平坦的分布，全局跳跃

**交换操作**：

在两个链之间交换状态：

$$
r(x_i, x_j) = \min\left(1, \frac{f_i(x_j)f_j(x_i)}{f_i(x_i)f_j(x_j)}\right)
$$

**目标分布**：

- 可见性分布：$p^*_{vis} = \mathbb{1}_{C(x) > 0}$
- 原始贡献分布：$p^*_{con} = \text{lum}(C(x))$

**组合权重**：

$$
w_{MCMC}(x, x_{light}) = \frac{p^*_c(C)/b_c}{p^*_{vis}(C)/b_{vis} + p^*_{con}(C)/b_{con}}
$$

### 7.5.2 AMCMCVCM/UPS算法

**基本框架**：

将MCMC算法作为光源子路径的采样器，与VCM/UPS框架结合。

**摄像机子路径**：使用传统MC算法（均匀分布）
**光源子路径**：使用MCMC算法（自适应分布）

**连接方式**：

1. 生成摄像机子路径，存储所有顶点
2. 使用MCMC生成光源子路径
3. 通过VCM/UPS框架连接/合并子路径

**自适应步幅**：

$$
u_{i+1} = \nu_i + \frac{A_{i+1} - A^*}{i}
$$

其中：
- $A^* = 0.234$：目标接受率
- $A_{i+1}$：当前接受率

**光源子路径贡献**：

$$
C(x_{light}) = \sum_{(x,t) \in P(x_{light})} \frac{f(x)w_t(x)}{p_t(x)}
$$

### 7.5.3 算法优势

**MC算法的优点**：
- 样本独立，全局分布均匀
- 适合处理直接光照
- 易于并行化

**MCMC算法的优点**：
- 探索局部区域高效
- 适合处理困难路径（SDS、复杂可见性）
- 样本利用率高

**组合优势**：
- 兼顾全局和局部探索
- 适应各种场景类型
- 更快的收敛速度
