# Ch.11 基于体素的全局光照

> **章节目标**：理解体素化表示方法，掌握圆锥追踪技术，了解基于体素的动态全局光照实现。

## 11.1 圆锥体内的光线追踪

### 11.1.1 光线追踪的问题

**传统光线追踪的局限**：

| 问题 | 原因 | 影响 |
|------|------|------|
| 走样 | 光线离开像素后失去联系 | 锯齿边缘 |
| 超采样成本高 | 方差大的像素需要更多光线 | 性能下降 |
| 忽略细节 | 无法获知采样点周围信息 | 丢失高频细节 |

### 11.1.2 圆锥体追踪原理

**核心思想**：将像素视为"面积"而非"点"，使用圆锥体作为光线定义。

```
光线追踪 vs 圆锥体追踪
────────────────────────

光线追踪：                    圆锥体追踪：
  ───────>                       /\
  起点+方向                    /  \
                              /____\
                              起点+圆锥角
```

**关键区别**：

1. **相交结果**：
   - 光线追踪：点
   - 圆锥体追踪：面积或体积

2. **反走样能力**：
   - 光线追踪：需要超采样
   - 圆锥体追踪：部分覆盖信息直接提供反走样

### 11.1.3 体积样本采样

**体积样本（Volume Samples）**：

每个体积样本返回两个值：

```
体积样本数据结构
────────────────
struct VolumeSample {
    float3 radiance;    // 光照估计
    float visibility;   // 遮挡比例（类似透明度）
};
```

**累积方式**：

类似于参与介质中的光线步进：

```
累积算法
────────────────
accumulated_radiance = 0
accumulated_transmittance = 1

for each volume_sample along cone:
    // 累加光照
    accumulated_radiance += 
        sample.radiance * accumulated_transmittance
    
    // 更新透明度
    accumulated_transmittance *= sample.visibility
```

### 11.1.4 精确度问题

**透明度混合误差**：

当多个物体部分遮挡圆锥体时，可能出现不正确的可见性计算：

```
误差示例
────────────────
实际：物体A遮挡50%，物体B遮挡50%
      → 总遮挡 = 100%（完全遮挡）

圆锥体追踪：透明度混合
      → 可见性 = 50% × 50% = 25%（错误！）
```

**结论**：圆锥体追踪是一种粗略近似，但在实践中效果可接受。

## 11.2 体积化的几何表述

### 11.2.1 体素预过滤

**核心思想**：将表面参数转化为体积密度分布，实现线性预过滤。

```
预过滤的优势
────────────────

传统多边形：                体素表述：
表面A ──┐                   体素网格
       ├── 无法线性组合     ┌───┬───┐
表面B ──┘                   │ A │ B │ 可线性平均
                            └───┴───┘
```

**体素（Voxel）**：

3D空间中的"像素"，存储体积数据：

| 存储数据 | 用途 |
|---------|------|
| 透明度 | 可见性计算 |
| 漫反射颜色 | 材质着色 |
| 法线分布 | 反射方向 |
| 辐射亮度 | 间接光照 |

### 11.2.2 基于预积分的圆锥体追踪

**体积渲染积分**：

$$
I(D) = I_0 T(s_0, D) + \int_{s_0}^{D} q(s)T(s, D)ds
$$

其中：
- $T(s_1, s_2) = e^{-\tau(s_1, s_2)}$：透明度
- $\tau(s_1, s_2) = \int_{s_1}^{s_2} \kappa(t)dt$：光学深度

**预积分模型**：

将积分拆分为离散区间的组合：

$$
I(D, P) \approx \sum_{i=0}^{n} Q_i \cdot \prod_{j=i+1}^{n} T_j
$$

其中：
- $Q_i = Q(s_i, s_{i+1}, P)$：平均向内散射能量
- $T_j = T(s_j, s_{j+1}, P)$：平均透明度

**增量式组合方案**：

```
算法：增量式光照累积
────────────────
Îₙ = Qₙ          // 初始能量
T̂ₙ = Tₙ          // 初始透明度

for i = n-1 downto 0:
    Îᵢ = Îᵢ₊₁ + T̂ᵢ₊₁ · Qᵢ    // 累积能量
    T̂ᵢ = T̂ᵢ₊₁ · Tᵢ           // 累积透明度

return Î₀         // 最终能量
```

### 11.2.3 多级体素结构

**八叉树层次**：

```
多级体素结构
────────────────

Level 0: 最精细（叶节点）
    ┌───┬───┬───┬───┐
    │   │   │   │   │
    ├───┼───┼───┼───┤
    │   │   │   │   │
    └───┴───┴───┴───┘

Level 1: 中等分辨率
    ┌───────┬───────┐
    │       │       │
    ├───────┼───────┤
    │       │       │
    └───────┴───────┘

Level 2: 最粗糙（根节点）
    ┌───────────────┐
    │               │
    └───────────────┘
```

**四线性插值（Quadrilinear Interpolation）**：

```
四线性插值维度
────────────────
1. x方向（空间）
2. y方向（空间）
3. z方向（空间）
4. LOD方向（层级）
```

**圆锥体足迹近似**：

```
圆锥体采样策略
────────────────

光线方向 ────────>
    │
    ├── 位置p处采样
    │   └── 使用面积s = πr²选择LOD
    │
    ├── 位置p'处采样
    │   └── 使用面积s' = π(r')²选择LOD
    │
    └── 累加所有样本贡献
```

### 11.2.4 预过滤的着色参数

**材质参数与线性关系**：

| 参数 | 与着色结果关系 | 处理方式 |
|------|--------------|---------|
| 漫反射颜色 | 线性 | 预过滤 |
| 法线分布 | 非线性→线性近似 | 预过滤 |
| 透明度 | 非线性 | 预积分 |
| BRDF | 非线性 | 预积分 |

#### 11.2.4.1 漫反射颜色

**预过滤方法**：

直接对体素内所有表面的漫反射颜色取平均：

$$
C_{RGB} = \frac{1}{N} \sum_{i=1}^{N} c_i
$$

构建多级纹理结构。

#### 11.2.4.2 法线方向信息

**法线分布函数（NDF）**：

存储体素内表面法线的方向分布。

**Toksvig近似**[Toksvig, 2005]**：**

使用平均法线长度作为高斯分布标准偏差：

$$
\sigma^2 = \frac{1 - |N|}{|N|}
$$

其中 $|N|$ 是平均法线的范数。

**粗糙度推导**：

```
平均法线长度 → 粗糙度
────────────────
|N| = 1.0  → σ = 0    （完美光滑）
|N| = 0.5  → σ = 1.0  （中等粗糙）
|N| → 0    → σ → ∞   （完全漫反射）
```

### 11.2.5 透明度和BRDF分布

**透明度预积分**：

计算从某个方向看穿体素的透明度：

$$
T(p, d, s, l) = \int_{r \in P} e^{-\tau(s, s+l, r)} dr
$$

**BRDF预积分**：

对于光泽表面，存储预积分的BRDF分布。

## 11.3 数据结构

### 11.3.1 稀疏体素八叉树（Sparse Voxel Octree）

**数据结构**：

```
节点结构
────────────────
struct OctreeNode {
    // 体素数据
    float3 albedo;          // 漫反射颜色
    float3 normal;          // 平均法线
    float opacity;          // 不透明度
    float3 radiance;        // 辐射亮度（自发光+反射）
    
    // 子节点指针（稀疏存储）
    uint childPointer;      // 指向8个子节点
    uint flags;             // 节点类型标记
};
```

**稀疏存储**：

仅存储包含几何体的体素，空体素不分配内存。

### 11.3.2 体素化流程

**GPU体素化**：

```
体素化算法
────────────────
1. 正交投影渲染场景到3D纹理
2. 每个片段写入对应体素位置
3. 保守光栅化确保覆盖
4. 构建八叉树层次结构
5. 过滤生成多级分辨率
```

**保守光栅化**：

确保三角形覆盖的所有体素都被标记，不错过小几何体。

### 11.3.3 动态更新

**部分更新策略**：

```
动态体素更新
────────────────
1. 标记变化区域
2. 重新体素化受影响区域
3. 更新八叉树节点
4. 重新过滤多级分辨率
```

## 11.4 渲染

### 11.4.1 间接漫反射

**算法流程**：

```
间接漫反射计算
────────────────
对于每个像素：
1. 从着色点发射多个圆锥体（覆盖半球）
2. 对每个圆锥体：
   a. 沿圆锥体方向步进
   b. 在每个体素采样点：
      - 根据圆锥体截面面积选择LOD
      - 四线性插值获取体素数据
      - 计算局部着色贡献
      - 累积透明度和光照
3. 累加所有圆锥体贡献
4. 乘以BRDF和余弦项
```

**圆锥体分布**：

```
半球圆锥体分布
────────────────
      /\
     /  \
    /____\    主方向（法线）
   /\    /\
  /  \  /  \   次要方向（余弦加权）
 /____\/____\
```

### 11.4.2 光泽反射

**窄圆锥体追踪**：

根据粗糙度确定圆锥体角度：

```
粗糙度与圆锥角
────────────────
粗糙度 = 0.0  → 圆锥角 = 0°    （完美镜面）
粗糙度 = 0.5  → 圆锥角 = 30°   （中等光泽）
粗糙度 = 1.0  → 圆锥角 = 90°   （漫反射）
```

### 11.4.3 环境光遮挡

**AO圆锥体**：

```
AO计算
────────────────
对于每个着色点：
1. 发射多个半球圆锥体
2. 对每个圆锥体追踪：
   - 累积透明度
   - 权重随距离衰减
3. AO = 1 - 平均遮挡
```

### 11.4.4 软阴影

**光源圆锥体追踪**：

```
软阴影计算
────────────────
1. 从着色点向光源方向发射圆锥体
2. 圆锥体角度 = 光源大小 / 距离
3. 追踪过程中累积不透明度
4. 阴影强度 = 累积不透明度
```
