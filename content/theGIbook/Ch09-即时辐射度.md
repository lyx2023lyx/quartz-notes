> **章节目标**：理解即时辐射度的核心思想，掌握VPL生成和渲染流程，了解反射阴影图等优化技术。

## 9.1 基本算法

### 9.1.1 核心思想

**与辐射度方法的区别**：

| 特性 | 传统辐射度 | 即时辐射度 |
|------|-----------|-----------|
| 离散化对象 | 曲面片 | 虚拟点光源 |
| 可见性 | 摄像机直接看到离散化 | 虚拟点光源不可见 |
| 人工痕迹 | 明显（边界、插值） | 平滑无痕迹 |
| 计算效率 | 较低 | 高（GPU光栅化） |

**核心原理**：

将所有入射辐射亮度 $L(y)$ 近似为 $M$ 个虚拟点光源的加和：

$$
L(y) \approx \sum_{i=0}^{M-1} L_i \delta(y - P_i)
$$

其中：
- $L_i$：第 $i$ 个虚拟点光源的辐射亮度
- $P_i$：第 $i$ 个虚拟点光源的位置
- $\delta$：狄拉克函数

**渲染方程的近似**：

$$
L_{mn} \approx \langle L_e, \omega_{mn} \rangle + \sum_{i=0}^{M-1} T_{mn} L_i \delta(y - P_i)
$$

**白话解读**：
- 间接光照被转化为大量"点光源"的直接光照计算
- 每个像素同时被所有虚拟点光源照射
- 计算可以高度并行化（GPU友好）

### 9.1.2 路径分解

在即时辐射度中，全路径 $\bar{x} = (x_0, x_1, \cdots, x_n)$ 被分解为三部分：

```
路径分解示意图
────────────────

光源 ──xV── xS ──xS── ... ──xC── 摄像机
        ↑      ↑            ↑
    虚拟点    余下        摄像机子路径
    光源     光源子路径    (通常为1)
```

**各部分的含义**：
1. $\bar{x}_C$：摄像机子路径，$x_0$ 位于摄像机焦点
2. $x_V$：虚拟点光源，光源子路径的末点
3. $\bar{x}_S$：余下光源子路径顶点

### 9.1.3 粒子生成与拟随机行走

**粒子 vs 光子**：

| 特性 | 光子映射 | 即时辐射度 |
|------|---------|-----------|
| 表面交互 | 可经过光泽/镜面 | 仅漫反射 |
| 能量处理 | 俄罗斯轮盘 | 保留真实能量 |
| 计算方式 | 密度估计 | 点光源计算 |

**粒子生成原理**：

使用霍尔顿序列（Halton sequence）生成拟随机数：

$$
x_i = (\Phi_{b_1}(i), \cdots, \Phi_{b_s}(i))
$$

其中 $b_j$ 表示第 $j$ 个素数。

**诺依曼级数**：

递归渲染方程可写为：

$$
L = (I - T_{fr})^{-1} L_e = \sum_{j=0}^{\infty} T_{fr}^j L_e
$$

**部分吸收（Fractional Absorption）**：

使用漫反射系数的平均值 $\rho$ 决定粒子存活：

$$
\rho := \frac{\sum_{k=1}^K \rho_{d,k}|A_k|}{\sum_{k=1}^K |A_k|} \approx ||T_{fd}||
$$

**总粒子数量**：

$$
M < \sum_{j=0}^{\infty} \rho^j N = \frac{1}{1-\rho}N =: lN
$$

其中 $l$ 表示平均路径长度。

### 9.1.4 算法流程

```
算法：即时辐射度
────────────────
输入：粒子数量 N，漫反射系数 ρ
输出：渲染图像

void InstantRadiosity(int N, double ρ) {
    Start = End = N;
    Reflections = 0;
    
    while (End > 0) {
        Start *= ρ;
        
        for (i = Start; i < End; i++) {
            // 选择光源起点位置
            y = y₀(Φ₂(i), Φ₃(i));
            L = Le(y) * supp(Le);
            
            // 粒子的多次反射
            for (j = 0; j <= Reflections; j++) {
                // 渲染带阴影的场景
                glRenderShadowedScene(L, y);
                glAccum(GL_ACCUM, 1/N);
                
                // 产生漫反射方向
                ω⃗ = ω⃗d(Φ_{b_{2j+2}}(i), Φ_{b_{2j+3}}(i));
                y = h(y, ω⃗);
                L *= fd(y);
            }
        }
        
        Reflections++;
        End = Start;
    }
    
    glAccum(GL_RETURN, 1.0);
}
```

**方向采样**：

使用霍尔顿序列分量转换为余弦分布方向：

$$
\vec{\omega} = \vec{\omega}_d(\Phi_{b_{2j+2}}, \Phi_{b_{2j+3}}) = (\arcsin\sqrt{\Phi_{b_{2j+2}}}, 2\pi\Phi_{b_{2j+3}})
$$

## 9.2 近似方法

### 9.2.1 反射阴影图（RSM）

**基本思想**[Dachsbacher and Stamminger, 2005]**：**

使用阴影图存储反射光照信息，每个像素作为一个小面积光源。

**数据存储**：

| 缓冲区 | 存储内容 | 用途 |
|--------|---------|------|
| 深度缓冲 | 深度值 $d_p$ | 阴影计算 |
| 法线缓冲 | 法线 $n_p$ | 余弦衰减 |
| 位置缓冲 | 世界坐标 $x_p$ | 距离计算 |
| 辐射通量缓冲 | $\Phi_p$ | 光照强度 |

**像素光源辐射强度**：

$$
I_p(\omega) = \Phi_p \max\{0, \langle n_p, \omega \rangle\}
$$

**辐射照度计算**：

$$
E_p(x, n) = \Phi_p \frac{\max\{0, \langle n_p, x - x_p \rangle\} \max\{0, \langle n, x_p - x \rangle\}}{||x - x_p||^4}
$$

**重要性采样**：

将点 $x$ 投影到阴影图位置 $(s,t)$，按距离平方反比采样：

$$
(s + r_{max}\xi_1\sin(2\pi\xi_2), t + r_{max}\xi_1\cos(2\pi\xi_2))
$$

**屏幕空间插值**：

```
三阶段渲染流程
────────────────
1. 低分辨率渲染 + 间接光照计算
2. 全分辨率渲染（插值检查法线和位置）
3. 对未插值像素执行完整VPL计算
```

### 9.2.2 溅射法（Splatting）

**改进点**[Dachsbacher and Stamminger, 2006]**：**

1. **发射而非收集**：虚拟点光源"溅射"光照到屏幕空间
2. **重要性采样**：阶层式变形（hierarchical warping）
3. **紧凑包围盒**：限制每个VPL的影响范围

**阶层式变形采样**：

```
阶层式变形过程
────────────────
1. 将阴影图划分为四个象限
2. 计算每个象限的辐射通量比例
3. 按比例挤压样本分布
4. 递归处理每个象限
```

**权重补偿**：

经过变形后，样本权重需要调整以保持总权重不变：

$$
w_i' = \begin{cases}
w_i/(2\Phi_r) & \text{if } y_i < \Phi_r \\
w_i/(2(1-\Phi_r)) & \text{otherwise}
\end{cases}
$$

**紧凑包围盒**：

使用椭球体近似像素光源影响区域：

$$
B(\alpha, r) = \frac{I_0 \cos(\alpha)^n}{r^2} = I_{low}
$$

### 9.2.3 不完全阴影图（ISM）

**核心思想**[Ritschel et al., 2008]**：**

间接光照频率变化平滑，可以使用低分辨率阴影图近似。

**预处理**：

将场景近似为点云表述，每个点存储：
- 所在三角形索引
- 质心坐标
- 法线和反射系数（可获取）

**抛物线映射（Parabolic Mapping）**：

覆盖整个半球面方向，区别于传统阴影图的正交/透视投影。

**推拉法（Pull-Push）**：

解决稀疏点采样导致的"漏洞"问题：

```
拉阶段（Pull）：
- 创建图像金字塔
- 对高分辨率图像下采样
- 仅使用"合法"像素平均

推阶段（Push）：
- 从低分辨率插值填充漏洞
- 重建合理的深度图
```

### 9.2.4 自适应不完全阴影图

**双向反射阴影图（BRSM）**[Ritschel et al., 2011]**：**

```
BRSM生成流程
────────────────
1. 渲染当前视图 → View Buffer
2. 生成反射阴影图 → RSM
3. 随机选择少量视图样本（~0.1%）
4. 计算每个pVPL对这些样本的贡献
5. 存储双向重要性到BRSM
6. 按重要性采样选择最终VPL
```

**自适应点采样**：

使用三角形相对于视图样本的立体角进行重要性采样：

$$
\text{采样概率} \propto \text{立体角}(V, \text{三角形}_i)
$$

## 9.3 适应性方法

### 9.3.1 拒绝不重要的样本

**方法**[Iliyan and Philipp, 2010]**：**

使用俄罗斯轮盘决定是否存储虚拟点光源：

$$
p_i = \min\left\{\frac{\Phi_i}{\Phi_v} + \epsilon, 1\right\}
$$

其中：
- $\Phi_i$：第 $i$ 个VPL对所有像素的贡献
- $\Phi_v$：所有VPL的平均贡献
- $\epsilon$：随机偏移（保证 $p_i > 0$）

**接受准则**：

若 $\xi \leq p_i$（$\xi \in [0,1)$ 均匀分布），则接受该VPL，并将其能量除以 $p_i$ 保持无偏。

### 9.3.2 双向即时辐射度

**核心思想**[Segovia et al., 2006]**：**

从摄像机反向行走生成"反向虚拟点光源"：

```
采样/重采样方法
────────────────
1. 生成标准VPL（光源→漫反射）
2. 生成反向VPL（摄像机→x₁→x₂）
3. 对每个VPL，从摄像机发射M条路径（M=10）
4. 计算VPL对视图的总贡献
5. 建立累积分布函数（CDF）
6. 按CDF重采样VPL
```

**概率密度估计**：

对于标准VPL：

$$
p(x_2) = \frac{1}{N} \sum_{i_1=1}^{N} p(x_{i_1} \to x_2)
$$

对于反向VPL：
- 已位于可视区域
- 仅需M=5个样本估计

### 9.3.3 梅特波利斯即时辐射度

**核心思想**[Segovia et al., 2007]**：**

将MLT算法集成到即时辐射度中，使VPL分布自动适配视图重要性。

**实现要点**：

1. **初始化**：使用双向路径追踪预计算图像估计，得到归一化系数 $P_c$
2. **MLT采样**：生成概率密度正比于真实光照的路径样本
3. **VPL提取**：
   - 每个路径样本对摄像机的总能量贡献为 $P_c/n$
   - 假设漫反射，均匀洒向各像素
   - 计算各像素接收的光照 $p_i'$
   - 缩放因子：$P_c/(n \cdot p_i')$

**限制**：
- 仅适用于辐射度设置（漫反射）
- 光泽表面的辐射亮度场难以表示

## 9.4 偏差补偿

### 9.4.1 问题分析

**几何项奇异值**：

当VPL与着色点距离很近时，几何项产生奇异值：

$$
G(x \leftrightarrow y) = \frac{\cos\theta_x \cos\theta_y}{||x-y||^2}
$$

**限定几何项**：

$$
G_b(x \leftrightarrow y) = \min\{G(x \leftrightarrow y), b\}
$$

**残留几何项**：

$$
G_r(x \leftrightarrow y) = G(x \leftrightarrow y) - G_b(x \leftrightarrow y) = \max\{G(x \leftrightarrow y) - b, 0\}
$$

### 9.4.2 光线追踪补偿法

**核心思想**[Kollig and Keller, 2004]**：**

将残留几何项的光照计算退化为传统光线追踪。

**积分转换**：

$$
\begin{aligned}
& L(x, \omega_r) - L_b(x, \omega_r) \\
=& \int_{S^2} L_e(h(x, \omega), -\omega) \frac{\max\{G(x, h(x, \omega)) - b, 0\}}{G(x, h(x, \omega))} f_r(\omega, x, \omega_r) \cos\theta_x d\omega
\end{aligned}
$$

**迭代过程**：

```
偏差补偿迭代
────────────────
着色点 x₁
  ├── 限定区域（Gb < b）：标准VPL直接光照（红色箭头）
  └── 残留区域（G > b）：光线追踪采样 → 补偿顶点 x₂
      
补偿顶点 x₂
  ├── 限定区域：标准VPL直接光照
  └── 残留区域：光线追踪采样 → 补偿顶点 x₃
      
... 迭代直到满足终止条件
```

**终止条件**：

使用俄罗斯轮盘采样决定光线的反射或吸收。

### 9.4.3 补偿方法的特性

**优点**：
- 无偏估计（保留所有能量）
- 残留区域计算成本低（范围小）
- 可以迭代处理多次反射

**缺点**：
- 迭代深度增加时退化为传统光线追踪
- 难以满足实时/交互式需求

## 9.5 避免奇异值——光泽表面的处理

### 9.5.1 光泽反射的挑战

**问题**：
1. 镜面/光泽表面的几何项奇异值更明显
2. 需要更高数量的VPL
3. 传统即时辐射度主要聚焦漫反射

### 9.5.2 光泽VPL的表示

**Phong模型**：

光泽VPL发射的辐射强度：

$$
I_p(\omega) = \Phi_p \max\{0, \langle r_p, \omega \rangle\}^P
$$

其中：
- $r_p$：主要反射方向
- $P$：Phong指数

### 9.5.3 紧凑包围盒

**椭球体近似**：

$$
B(\alpha, r) = \frac{I_0 \cos(\alpha)^n}{r^2} \leq I_{low}
$$

包围区域随Phong指数变化：
- $n=1$：漫反射（蛋形）
- $n=10$：中等光泽
- $n=100$：近似镜面（狭窄叶片）

## 9.6 可扩展性

### 9.6.1 计算复杂度分析

**标准即时辐射度**：
- VPL生成：$O(M)$
- 光照计算：$O(M \times P)$，$P$为像素数
- 总体：线性复杂度

**优化后**：
- 屏幕空间插值：$O(M \times P_{low})$
- 紧凑包围盒：$O(M_{active} \times P)$，$M_{active} \ll M$
- 自适应方法：$O(M_{effective} \times P)$

### 9.6.2 大规模场景处理

**分层处理**：

```
大规模场景优化
────────────────
1. 将场景划分为多个系统（System）
2. 每个系统内独立计算即时辐射度
3. 系统间通过光照探针传递光照
4. 远距离使用低分辨率近似
```

### 9.6.3 现代GPU实现

**计算着色器优化**：

- VPL生成：并行粒子追踪
- 光照计算：分块（tile-based）处理
- 阴影计算：级联不完全阴影图
