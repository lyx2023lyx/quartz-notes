# Ch.8 辐射度方法

> **章节目标**：掌握辐射度方法的理论基础，理解形状因子计算和有限元离散化，了解层级辐射度等高级技术。

---

## 8.1 基本概念

### 8.1.1 辐射度方法的起源

**历史背景**：
- 起源于与光线追踪同一时期 [Goral et al., 1984]
- 用于高效计算间接漫反射光照
- 适用于实时渲染中的动态光照计算

**核心假设**：
1. **漫反射表面**：出射光照与方向无关
2. **低频变化**：漫反射光照分布变化缓慢
3. **网格化**：场景可细分为多个曲面片

**与蒙特卡洛方法的对比**：

| 特性 | 辐射度方法 | 蒙特卡洛方法 |
|------|------------|--------------|
| 计算方式 | 确定性求解 | 随机采样 |
| 精度 | 近似但稳定 | 渐进收敛 |
| 速度 | 预计算+实时 | 离线渲染 |
| 适用场景 | 静态/准静态 | 复杂光照 |

### 8.1.2 辐射度方程

**渲染方程（漫反射简化）**：

对于纯漫反射表面，渲染方程简化为：

$$
L(x) = L_e(x) + \rho(x) \int_S K(x, y) L(y) dA_y
$$

其中：
- $L_e(x)$：自发光辐射亮度
- $\rho(x)$：漫反射系数
- $K(x, y) = G(x, y)V(x, y)$：核函数

**几何项**：

$$
G(x, y) = \frac{\cos(\omega_{xy}, N_x)\cos(-\omega_{xy}, N_y)}{\pi r_{xy}^2}
$$

**白话解读**：几何项描述了辐射亮度从点 $y$ 传输到点 $x$ 过程中的衰减，包含：
1. 两次余弦衰减（投影到垂直于传输方向的平面）
2. 距离平方反比衰减
3. $\pi$ 是漫反射BRDF的归一化参数

**辐射度定义**：

$$
B(x) = \pi L(x)
$$

辐射度 $B$ 表示单位面积上离开表面的总辐射能量。

### 8.1.3 辐射度线性方程组

**离散化**：

将表面划分为 $n$ 个曲面片，每个片内的辐射度近似为常数：

$$
B_i = B_i^e + \rho_i \sum_{j=1}^{n} F_{ij} B_j
$$

**形状系数（Form Factor）**：

$$
F_{ij} = \frac{1}{A_i} \int_{S_i} \int_{S_j} K(x, y) dA_y dA_x
$$

**形状系数的物理意义**：
- 表示曲面片 $j$ 向整个空间发射的辐射能量中，落于曲面片 $i$ 上的分数部分
- 纯几何量，与光照分布无关

**形状系数的性质**：

1. **非负性**：$F_{ij} \geq 0$
2. **归一性**：$\sum_{j} F_{ij} \leq 1$（闭合场景取等号）
3. **互反性**：$A_i F_{ij} = A_j F_{ji}$

**矩阵形式**：

$$
\mathbf{B} = \mathbf{B}^e + \mathbf{\rho} \mathbf{F} \mathbf{B}
$$

或：

$$
(\mathbf{I} - \mathbf{\rho}\mathbf{F})\mathbf{B} = \mathbf{B}^e
$$

其中：
- $\mathbf{B}$：辐射度向量 $(B_1, B_2, \cdots, B_n)^T$
- $\mathbf{B}^e$：自发光辐射度向量
- $\mathbf{\rho}$：对角矩阵 $\text{diag}(\rho_1, \rho_2, \cdots, \rho_n)$
- $\mathbf{F}$：形状系数矩阵

## 8.2 形状系数

### 8.2.1 半立方体方法（Hemi-cube）

**基本原理**：

[Cohen and Greenberg, 1985] 提出使用半立方体来计算形状系数。

**假设条件**：
- 两曲面片距离足够大
- 曲面片之间无遮挡

**近似公式**：

$$
F_{ij} \approx F_{ij}^\delta = \int_{S_j} \frac{\cos\omega_x \cos\omega_y}{\pi r_{xy}^2} HID \, dA_j
$$

其中 $HID$ 是可见性函数（1或0）。

**努塞尔类比（Nusselt Analog）**：

曲面 $j$ 对曲面 $i$ 的形状系数等于：

$$
F_{ij} = A_i \times \frac{\text{曲面}j\text{在半球上的投影面积}}{\text{半球面积}}
$$

**算法流程**：

```
算法：半立方体方法计算形状系数
────────────────────────────────
输入：源曲面片 i
输出：所有 F_ij

1. 在曲面 i 中心构造半立方体
2. 对半立方体每个面进行光栅化：
   - 放置摄像机于曲面中心
   - 渲染场景得到深度图
3. 对每个像素：
   - 确定可见曲面 j
   - 计算该像素到半球面的投影映射
   - 累加贡献到 F_ij
4. 归一化所有 F_ij
```

**投影映射表**：

预计算半立方体上每个像素到半球面的投影面积比，存储为查找表。

### 8.2.2 蒙特卡洛方法

**局部线段方法**[Sillion and Puech, 1989]**：**

**思路**：从曲面 $i$ 向半球面方向发射光线，统计落于曲面 $j$ 的比例。

**概率密度函数**：

$$
p(x, \omega, \psi) = \frac{\cos\omega \sin\omega}{\pi A_i}
$$

**估计公式**：

$$
F_{ij} \approx \frac{1}{N} \sum_{k=1}^{N} V_j(x_k, \omega_k, \psi_k)
$$

**白话解读**：
- 从曲面 $i$ 上随机位置向随机方向发射光线
- 统计有多少光线落在曲面 $j$ 上
- 比例即为形状系数的估计

**全局线段方法**[Sbert, 1993]**：**

**思路**：在场景包围球上随机发射全局光线，统计与曲面的相交情况。

**优势**：
- 内存连贯性好
- 易于并行化

**劣势**：
- 小曲面方差大
- 不能单独调整特定曲面的采样密度

### 8.2.3 形状系数计算的选择策略

| 方法 | 适用场景 | 优点 | 缺点 |
|------|----------|------|------|
| 半立方体 | 大规模场景 | GPU加速 | 精度受限 |
| 局部线段 | 精确计算 | 可重要性采样 | 计算量大 |
| 全局线段 | 均匀介质 | 内存连贯 | 小曲面误差大 |

## 8.3 求解线性方程组

### 8.3.1 直接法 vs 迭代法

**直接法（高斯消元）**：

- 计算复杂度：$O(n^3)$
- 优点：精确解
- 缺点：存储需求大，数值不稳定（取整误差累积）

**迭代法**：

- 计算复杂度：$O(kn^2)$，$k$ 为迭代次数
- 优点：内存友好，可逐步修正误差
- 缺点：收敛速度依赖矩阵性质

### 8.3.2 雅可比方法（Jacobi Method）

**基本思想**：

从初始猜测出发，逐步迭代逼近真实解。

**迭代公式**：

$$
B_i^{(k+1)} = B_i^e + \rho_i \sum_{j \neq i} F_{ij} B_j^{(k)}
$$

**算法流程**：

```
算法：雅可比迭代
────────────────
输入：初始猜测 B⁽⁰⁾，容差 ε
输出：辐射度向量 B

1. 初始化：k ← 0
2. 重复直到收敛：
   for i = 1 to n:
       B_i^{(k+1)} ← B_i^e + ρ_i Σ_{j≠i} F_{ij} B_j^{(k)}
   
   if ||B^{(k+1)} - B^{(k)}|| < ε:
       返回 B^{(k+1)}
   
   k ← k + 1
```

**收敛条件**：

矩阵 $\mathbf{\rho}\mathbf{F}$ 的谱半径小于1时收敛。

### 8.3.3 高斯-赛德尔方法（Gauss-Seidel）

**改进**：

使用**最新计算的值**立即更新后续计算。

**迭代公式**：

$$
B_i^{(k+1)} = B_i^e + \rho_i \left( \sum_{j < i} F_{ij} B_j^{(k+1)} + \sum_{j > i} F_{ij} B_j^{(k)} \right)
$$

**优势**：
- 收敛速度更快
- 同样的计算量，更好的精度

**示例**：

考虑方程组：
$$
\begin{cases}
5x_1 - 2x_2 + 3x_3 = -1 \\
-3x_1 + 9x_2 + x_3 = 2 \\
2x_1 - x_2 - 7x_3 = 3
\end{cases}
$$

高斯-赛德尔迭代序列收敛更快（见书中表8.2）。

### 8.3.4 渐进式改进方法

**核心思想**：

将辐射度传输转换为"发射"过程，每次迭代处理一个曲面。

**发射（Shooting）vs 收集（Gathering）**：

```
收集方式 (Gathering):
─────────────────────
Bi ← Bei + ρi Σj Fij Bj
需要等待所有 Bj 已知

发射方式 (Shooting):
────────────────────
对所有曲面 j: ΔBj ← ρj Bi Fij Ai/Aj
Bi 同时影响所有其他曲面
```

**未发射辐射度**：

$$
\Delta B_i = B_i^{\text{new}} - B_i^{\text{old}}
$$

**算法流程**：

```
算法：渐进式改进辐射度
──────────────────────
1. 初始化：
   for 每个曲面 i:
       Bi ← Bei
       ΔBi ← Bei

2. 迭代直到收敛：
   while max(ΔBi) > ε:
       // 选择未发射辐射度最大的曲面
       i ← argmax(ΔB)
       
       // 发射辐射度
       for 每个曲面 j:
           ΔRad ← ρj · ΔBi · Fij · Ai/Aj
           ΔBj ← ΔBj + ΔRad
           Bj ← Bj + ΔRad
       
       // 重置未发射辐射度
       ΔBi ← 0
```

**优势**：
- 每次迭代都有可显示的中间结果
- 内存需求低（不需要存储整个形状系数矩阵）
- 适合交互式预览

## 8.4 曲面细分

### 8.4.1 经典辐射度方法的局限

**问题**：
1. 固定分辨率细分，不考虑光照分布
2. 阴影边缘处理差（辐射度假设为常数）
3. 存储复杂度 $O(n^2)$

### 8.4.2 阶层式辐射度方法（Hierarchical Radiosity）

**核心思想**[Hanrahan et al., 1991]**：**

根据形状系数的大小自适应细分曲面。

**四叉树结构**：

```
阶层式细分示意图
────────────────

        原始曲面 (父节点)
        /    |    \    \
    东北  西北  东南  西南 (子节点)
    /|\   /|\   /|\   /|\
   ...  ...  ...  ... (更细分)
```

**细分准则**：

```
算法：阶层式细分
────────────────
Refine(p, q, F_eps, A_eps):
    Fpq ← FormFactorEstimate(p, q)
    Fqp ← FormFactorEstimate(q, p)
    
    if Fpq < F_eps and Fqp < F_eps:
        Link(p, q)  // 建立链接
    else:
        if Fpq > Fqp:
            if Subdiv(q, A_eps):
                // 细分 q 为 4 个子曲面
                Refine(p, q.ne, F_eps, A_eps)
                Refine(p, q.nw, F_eps, A_eps)
                Refine(p, q.se, F_eps, A_eps)
                Refine(p, q.sw, F_eps, A_eps)
            else:
                Link(p, q)
        else:
            // 类似地细分 p
```

**参数说明**：
- $F_{eps}$：形状系数阈值（精度控制）
- $A_{eps}$：最小面积阈值（防止无限细分）

**优势**：
- 远距曲面使用低分辨率（形状系数小）
- 近距曲面使用高分辨率
- 存储复杂度可降至 $O(n)$

### 8.4.3 小波辐射度方法（Wavelet Radiosity）

**数学基础**：

小波分析提供了一种多分辨率信号分析方法。

**哈尔小波（Haar Wavelet）**：

**尺度函数**：

$$
\phi(x) = \begin{cases} 1 & 0 \leq x < 1 \\ 0 & \text{其他} \end{cases}
$$

**小波函数**：

$$
\psi(x) = \phi(2x) - \phi(2x - 1)
$$

**多分辨率分析**：

$$
L^2(\mathbb{R}) = V_0 \oplus W_0 \oplus W_1 \oplus \cdots
$$

其中：
- $V_j$：第 $j$ 级尺度函数空间
- $W_j$：第 $j$ 级小波空间（细节函数）

**小波辐射度的优势**：

1. **消失矩（Vanishing Moments）**：
   - 哈尔小波：1阶消失矩
   - 高阶小波：更好的信号逼近

2. **系数稀疏性**：
   - 平坦区域系数接近0
   - 只需存储非零系数

3. **层次结构**：
   - 自然对应阶层式细分
   - 支持自适应细分

**公式推导**：

将辐射度函数 $B(x)$ 投影到小波基：

$$
B(x) = \sum_k a_{0,k} \phi_{0,k}(x) + \sum_{j \geq 0} \sum_k d_{j,k} \psi_{j,k}(x)
$$

其中：
- $a_{0,k}$：近似系数（尺度函数系数）
- $d_{j,k}$：细节系数（小波系数）

### 8.4.4 非连续网格化（Discontinuity Meshing）

**问题**：

阴影边缘导致辐射度分布的非连续性，固定细分无法准确捕捉。

**非连续边界类型**[Lischinski et al., 1992]**：**

| 类型 | 描述 | 光照变化 |
|------|------|----------|
| D0 | 硬阴影边界 | 常数跳变 |
| D1 | 半影边界（线性） | 线性变化 |
| D2 | 半影边界（一般） | 二次变化 |

**视觉事件（Visual Events）**：

**VE事件（Vertex-Edge）**：
- 由一个顶点和一个边构成的平面
- 所有穿过顶点和边的线段集合

**EEE事件（Edge-Edge-Edge）**：
- 由三条斜交边界构成的二次曲面
- 所有同时穿过三条边界线段的集合

**可见性骨骼（Visibility Skeleton）**[Durand et al., 1997]**：**

数据结构用于存储所有可能的视觉事件：

```
class Node:  // 线段极值（节点）
    adjacentArcs: List[Arc]
    Fup, Fdown: Face  // 上下曲面
    Pup, Pdown: Point3D  // 端点

class Arc:   // 线段条带（边）
    Nstart, Nend: Node
    tstart, tend: float
    Fup, Fdown: Face

class EV(Arc):  // VE条带
    e: Edge
    v: Vertex
```

**惰性小波变换（Lazy Wavelet Transform）**：

用于处理非规则细分：

```
ΔB = Bl - Σ c_i · B_ni
```

其中：
- $B_l$：位置 $l$ 处的真实采样值
- $c_i$：插值系数
- 第二项：低分辨率近似值

### 8.4.5 组合方法

**点收集（Point Gathering）**：

从顶点直接收集辐射度，避免从曲面推导：

$$
F_{v,Q} = \frac{1}{2\pi} \vec{N} \cdot \sum_i \frac{\vec{R}_i \times \vec{R}_{i+1}}{||\vec{R}_i \times \vec{R}_{i+1}||} \gamma_i
$$

**约束德劳内三角化（CDT）**：

将曲面细分为三角形，同时包含预置的可见性边界线。

## 8.5 Enlighten中间件

### 8.5.1 概述

**Enlighten** 是目前流行的基于辐射度的实时全局光照解决方案，已集成到Unity等引擎。

**三大光照分量**：

```
实时渲染光照组成
────────────────

1. 直接光照（Direct Lighting）
   - 光栅化 + 阴影贴图
   
2. 间接光照（Indirect Lighting）
   - Enlighten处理
   - 光照贴图（Light Map）
   
3. 光泽反射（Specular Lighting）
   - 环境贴图
```

### 8.5.2 预处理阶段

**1. 打包（Packing）**：

- 将物体表面的纹理坐标打包成图表（Chart）
- 多个图表构成一个系统（System）
- 图表可合并以简化计算

**参数**：
- `Min Chart Size`：最小图表尺寸（默认4×4）
- `Scale In Lightmap`：单物体分辨率调节
- `Auto UV Max Distance`：合并距离阈值
- `Auto UV Max Angle`：合并角度阈值（默认89°）

**为什么图表尺寸影响纹素数量？**

每个图表至少占据 2×2 纹素，边缘预留半像素防止重叠。

**2. 聚类（Clustering）**：

- 将几何场景细分为簇（Cluster）
- 簇尺寸 = 光照贴图分辨率 × `Cluster Resolution`（默认0.5）

**3. 形状系数计算**：

- 使用局部线段方法
- 参数 `Irradiance Quality`：每像素光线数量（默认8192）
- 参数 `Irradiance Budget`：每簇存储的形状系数数量（默认128）

### 8.5.3 实时计算阶段

**输入**：
- 预处理数据（形状系数）
- 动态光照信息
- 材质属性（漫反射系数等）

**输出**：
- 光照贴图（间接漫反射）
- 光照探针（Light Probe）
- 反射捕捉（Reflection Capture）

**求解过程**：

```
1. 对直接光照结果进行点采样
2. 使用形状系数构建辐射度线性方程组
3. 迭代求解（每帧处理一个系统）
4. 双线性插值得到最终光照贴图
```

**线性外推**：

将曲面辐射度值外推到顶点：

$$
y = y_{k-1} + \frac{x - x_{k-1}}{x_k - x_{k-1}}(y_k - y_{k-1})
$$

## 8.6 实时动态辐射度方法

### 8.6.1 增量式辐射度方法

**核心思想**[Chen, 1990]**：**

场景修改时，仅计算辐射度增量，而非重新计算全部。

**表面属性修改**：

$$
\Delta B_i = (B_i^{e'} - B_i^e) + \frac{(\rho_i' - \rho_i)(B_i - B_i^e)}{\rho_i}
$$

**白话解读**：
- 第一项：自发光属性修改
- 第二项：漫反射系数修改导致的间接光照变化

### 8.6.2 辐射度重分配

**时间相关变量**[George et al., 1990]**：**

$$
B_i^t = B_i^e + \rho_i \sum_j F_{ij}^t B_j^t
$$

**辐射度增量**：

$$
B_i^{\Delta t} = R_i + \rho_i \sum_j F_{ij}^t B_j^{\Delta t}
$$

其中重分配项：

$$
R_i = \rho_i \sum_j F_{ij}^{\Delta t} B_j^{t-1}
$$

**白话解读**：
- 几何修改导致形状系数变化 $F_{ij}^{\Delta t}$
- 重分配项计算由于形状系数变化导致的辐射度重新分布

### 8.6.3 交叉重分配方法

**问题分析**[Hoef, 2013]**：**

传统方法需要对每个静态曲面渲染半立方体，计算复杂度高。

**解决方案**：

仅针对动态曲面渲染半立方体，静态曲面之间的重分配通过动态曲面"交叉"传递。

**重分配函数分类**：

| 类型 | 描述 | 计算方法 |
|------|------|----------|
| R(C,M) | 静态→动态 | 传统方法 |
| R(M,C) | 动态→静态 | 渐进式 |
| R(M,M) | 动态→动态 | 渐进式 |
| R(C,C) | 静态→静态 | 交叉重分配 |

**交叉投影**：

```
静态曲面 r,g,b → 动态曲面 j → 静态曲面 i

F_ij = F_ir + F_ig + F_ib
```

**复杂度**：

- 传统方法：$O(2|M| + 2|C|)$ 个半立方体
- 交叉方法：$O(4|M|)$ 个半立方体
- 通常 $|M| \ll |C|$，效率大幅提升
