## 1.1 什么是全局光照

### 核心区分

```
┌─────────────────────────────────────────────────────────────┐
│                     光照模型分类                            │
├─────────────────────────────────────────────────────────────┤
│                                                              │
│   局部光照(Local Illumination)                               │
│   ├── 仅考虑光源→表面→眼睛的一次反射                        │
│   └── 不考虑其他表面的反射影响                               │
│                                                              │
│   全局光照(Global Illumination)                              │
│   ├── 考虑光在场景中多次反弹                                 │
│   ├── 包含间接光照、颜色渗透                                 │
│   └── 更符合物理真实                                         │
│                                                              │
└─────────────────────────────────────────────────────────────┘
```

### 全局光照应实现的光学现象

| 现象 | 描述 | 难点 |
|------|------|------|
| **阴影(Shadows)** | 物体遮挡光源产生的暗区 | 软阴影需要计算半影区 |
| **环境遮挡(AO)** | 环境光被周围几何体遮挡的效果 | 需要对整个半球积分 |
| **反射(Reflection)** | 光滑表面对环境的镜像 | 需要知道周围环境的光照 |
| **间接光照(Indirect)** | 光经过多次反弹后的照明 | 颜色渗透效果 |
| **焦散(Caustics)** | 光通过镜面反射/折射汇聚 | 采样困难，路径概率低 |
| **次表面散射(SSS)** | 光进入物体内部散射后出射 | 需要模拟体积内的传播 |

## 1.2 辐射度量学

### 五大基本度量一览

| 名称 | 符号 | 单位 | 含义 | 记忆口诀 |
|------|------|------|------|---------|
| **辐射能量** | Q | J(焦耳) | 光子的总能量 | "总量" |
| **辐射通量** | Φ | W(瓦特) | 每秒发射的功率 | "功率=能量/时间" |
| **辐射照度** | E | W/m² | 表面接收的光照密度 | "入射=照射" |
| **辐射强度** | I | W/sr | 光源在特定方向的强度 | "光源方向性" |
| **辐射亮度** | L | W/(m²·sr) | 单束光的亮度 | "最终看到的" |

### 关键概念：立体角

**定义**：3D空间中从某点观察物体"有多大"的度量

```
                    ╱╲
                   ╱  ╲
                  ╱    ╲
                 ╱  A   ╲    A: 球面上的区域
                ╱________╲
               ╱∠Ω      ╲
              ●───────────    ●: 观察点(球心)
```

**公式**：
$$
\Omega = \iint_A \sin\theta \, d\theta \, d\varphi
$$

**解读**：
- 整个球面的立体角 = **4π sr**
- 立方体一个面从中心看的立体角 = **2π/3 sr**

### 核心公式详解

#### 1. 辐射亮度定义式（最重要）

$$
d\Phi = L \cos\theta \, dA \, d\omega
$$

**白话解读**：
- **dΦ**：通过某面元、某方向的微小能量
- **L**：这就是我们要计算的辐射亮度！
- **cosθ**：面元与光线方向的夹角余弦（越斜越弱）
- **dA**：面元面积
- **dω**：立体角

> **为什么重要**：渲染方程最终要求的就是 **L**！

#### 2. 辐射照度积分式

$$
E = \int L \cos\theta \, d\omega
$$

**白话解读**：
- 表面接收的总光照 = 所有入射方向的辐射亮度 × 余弦因子 的积分
- 这就是渲染方程中的积分项！

#### 3. 平方反比定律

$$
E = \frac{I \cos\theta}{r^2}
$$

**白话解读**：
- 光照强度随距离平方衰减
- 这就是为什么远处的灯看起来更暗
- **r**：到光源的距离

```
        💡 光源(I)
         │
         │
         │ r
         │
         ▼
    ┌─────────┐
    │  表面   │  E = I·cosθ / r²
    └─────────┘
```

## 1.3 物体的表面着色

### 几何光学三大假设

1. **表面绝对光滑**（微观粗糙度用统计方法处理）
2. **光仅能被发射、反射、传播**
3. **光速无限大，沿直线传播**

### 反射定律与折射定律

```
           入射光 l
              │
              │ θ₁
              ▼
    ──────────●──────────  表面
             ╱│
           ╱  │
         ╱ θ₁│ θ₂
       ╱─────┼─────▶ 反射光 r
              │
              │
              ▼ 折射光 t
```

**反射定律**：入射角 = 反射角，三线共面

**斯涅尔定律（折射）**：
$$
n_1 \sin\theta_1 = n_2 \sin\theta_2
$$

### 光源类型

| 类型 | 特点 | 应用 |
|------|------|------|
| **平行光** | 方向统一，无衰减 | 太阳光 |
| **点光源** | 向四面八方发射 | 灯泡 |
| **聚光灯** | 有方向限制 | 手电筒 |
| **环境光** | 来自整个半球 | 天光、间接光 |

## 1.4 采样和反走样

### 核心问题

> 数字图像是**离散**的，但真实世界是**连续**的。采样会丢失信息，走样是采样不足的必然结果。

### 傅里叶变换（一句话理解）

**作用**：将函数从"时间/空间域"转换到"频率域"

```
空间域                    频率域
   │                        │
   │    傅里叶变换           │
   │ ═══════════════════▶   │
   │                        │
   ▼                        ▼
 f(x,y)图像              F(μ)频率分布
```

**关键结论**：
- 频率域显示"变化有多快"
- **带限函数**：存在最大频率，超过后频率为0

### 采样定理（奈奎斯特定理）

> **要完美重建一个信号，采样率必须 ≥ 2 × 最高频率**

```
过采样      临界采样      欠采样（走样）
  │           │            │
  ▼           ▼            ▼
┌────┐     ┌────┐      ┌────┐
│    │     │    │      │╲╱╲╱│ ← 频谱重叠
│    │     │╲  ╱│      │╱╲╱╲│
└────┘     └────┘      └────┘
  OK         OK         失败
```

### 走样类型

| 类型 | 原因 | 解决方案 |
|------|------|---------|
| **几何走样** | 三角形边缘的可见性函数不连续 | MSAA、超采样 |
| **着色走样** | 光泽BRDF高频（粗糙度低） | 纹理过滤、预积分 |
| **纹理走样** | 纹理缩小或放大 | Mipmap、各向异性过滤 |
| **时间走样** | 运动物体采样不足 | 运动模糊 |

### 反走样技术

```
MSAA（多重采样反走样）
┌─────────────────┐
│  ·  ○  ·  ○    │  · = 子采样点位置
│                 │  ○ = 被几何覆盖的子采样点
│  ○  ·  ○  ·    │  
│                 │  每个像素：
│  ·  ○  ·  ○    │  - 多个子采样点计算深度/覆盖率
│                 │  - 但只计算一次颜色
│  ○  ·  ○  ·    │  - 减少计算量但保持边缘质量
└─────────────────┘
```

## 1.5 基于物理的渲染

### BRDF：双向反射分布函数

**定义**：描述光如何从入射方向反射到出射方向

$$
f_r(\omega_i, \omega_r) = \frac{dL_r(\omega_r)}{dE_i(\omega_i)} = \frac{dL_r(\omega_r)}{L_i(\omega_i)\cos\theta_i \, d\omega_i}
$$

**白话解读**：
- BRDF值 = 出射亮度增量 / 入射照度增量
- 简单说：**入射光有多少比例被反射到观察方向**

```
              ωr (观察方向)
              ▲
             ╱│
            ╱ │
           ╱  │
          ╱   │
    ─────●────┼─────  表面
        ╱╲   ╱
       ╱  ╲ ╱
      ╱    ●
     ╱    ╱  入射方向 ωi
    ╱    ╱
```

### BRDF两大物理性质

| 性质 | 公式 | 含义 |
|------|------|------|
| **互反律** | $f_r(\omega_i, \omega_r) = f_r(\omega_r, \omega_i)$ | 入射和出射可互换 |
| **能量守恒** | $\int f_r \cos\theta_i \, d\omega_i \leq 1$ | 反射光 ≤ 入射光 |

### 菲涅耳公式

**核心思想**：反射率取决于**入射角**和**折射率**

```
金属：                              非金属（绝缘体）：
┌────────┐                        ┌────────┐
│████████│ 几乎恒定的高反射率      │░░░░░░░░│ 低角度低反射
│████████│ (~80-100%)             │▓▓▓▓▓▓▓▓│ 高角度高反射
│████████│                        │████████│ (~100% at 90°)
└────────┘                        └────────┘
 0° → 90°                          0° → 90°
```

**Schlick近似**（工业常用）：
$$
R_F(\theta_i) \approx R_F(0°) + (1 - R_F(0°))(1 - \cos\theta_i)^5
$$

**解读**：
- $R_F(0°)$：垂直入射时的反射率（材质关键参数）
- 非金属的$R_F(0°)$约0.02-0.05（很暗）
- 金属的$R_F(0°)$约0.5-1.0（很亮）

### 微面元理论

**核心假设**：宏观表面由大量**微观光滑面元**组成

```
宏观表面（像素级别）
┌──────────────────────────────────────┐
│    ╱╲    ┌─┐    ╱╲    ╱╲             │
│   ╱  ╲   │ │   ╱  ╲  ╱  ╲            │
│  ╱ h ╲   └─┘  ╱    ╲╱    ╲           │
│ ╱──────╲     ╱            ╲          │
│           h = 半矢量 = (l+v)/|l+v|    │
└──────────────────────────────────────┘
       ↓
只有法线朝向h的微面元才能将l反射到v
```

#### 微面元BRDF公式

$$
f(l, v) = f_d + \frac{R_F(\theta_h) G(\theta_l, \theta_v) D(\theta_h)}{4\cos\theta_l \cos\theta_v}
$$

**各组件解读**：

| 组件 | 名称 | 作用 |
|------|------|------|
| $f_d$ | 漫反射项 | 与方向无关的基础颜色 |
| $R_F$ | 菲涅耳反射率 | 决定反射多少光 |
| $G$ | 几何遮挡函数 | 考虑微面元相互遮挡 |
| $D$ | 法线分布函数 | 描述微面元朝向分布 |
| 分母 | 转换因子 | 微观→宏观空间的转换 |

#### 主流法线分布函数(NDF)

| 名称 | 特点 | 应用 |
|------|------|------|
| **GGX** | 长尾分布，更接近真实材质 | UE4、Disney |
| **Beckmann** | 传统高斯分布 | 早期渲染器 |
| **GTR** | 可调指数，Disney使用双叶 | 电影级渲染 |

**GTR公式**：
$$
D_{GTR} = c / (\alpha^2 \cos^2\theta_h + \sin^2\theta_h)^\lambda
$$


## 1.6 渲染方程

### 最终形式

$$
L(x \to \omega_o) = L_e(x \to \omega_o) + \int_{\Omega_x} f_r(x, \omega_i \to \omega_o) L(x \leftarrow \omega_i) \cos(N_x, \omega_i) \, d\omega_i
$$

### 逐项解读

```
┌────────────────────────────────────────────────────────────────────┐
│                        渲染方程结构                                 │
├────────────────────────────────────────────────────────────────────┤
│                                                                    │
│  L(x → ωo) = Le(x → ωo)  +  ∫ fr · Li · cosθ dω                  │
│       │           │              │     │     │    │               │
│       │           │              │     │     │    └─ 积分         │
│       │           │              │     │     └────── 余弦因子     │
│       │           │              │     └──────────── 入射光       │
│       │           │              └────────────────── BRDF          │
│       │           └───────────────────────────────── 自发光       │
│       └───────────────────────────────────────────── 总出射光     │
│                                                                    │
└────────────────────────────────────────────────────────────────────┘
```

| 符号 | 含义 |
|------|------|
| $L(x \to \omega_o)$ | 点x沿出射方向ωo的总辐射亮度 |
| $L_e(x \to \omega_o)$ | 点x沿出射方向ωo的自发光 |
| $f_r$ | BRDF（双向反射分布函数） |
| $L(x \leftarrow \omega_i)$ | 从入射方向ωi到达点x的辐射亮度 |
| $\cos(N_x, \omega_i)$ | 法线与入射方向夹角的余弦 |
| $\Omega_x$ | 点x法线方向的半球 |

### 为什么渲染方程难算？

**原因1：递归性**
- L同时出现在等式左右两边
- 要算一个点的L，需要知道其他点的L

**原因2：积分维度高**
- 半球积分 = 2D积分
- 光路可能多次反弹 → 高维积分

**原因3：复杂性**
- 几何可见性
- 材质BRDF
- 光源分布

### 光线路径表达式

| 符号 | 含义 | 示例 |
|------|------|------|
| L | 光源 | - |
| E | 摄像机/眼睛 | - |
| D | 漫反射 | - |
| S | 光泽反射 | - |
| V | 体积散射 | - |
| * | 0次或多次 | - |
| + | 1次或多次 | - |

**常见路径**：
- $LE$：光源直射摄像机
- $L(D|S)E$：直接光照（一次反弹）
- $L(D|S)(D|S)^+E$：间接光照（两次及以上反弹）
- $LS^+E$：镜面反射路径
- $LD^+E$：漫反射路径（焦散）

## 1.7 离线与实时渲染

### 关键观点

> **不必刻意区分离线和实时渲染，应该追求掌握统一的图形学知识体系。**

**原因**：
1. 实时渲染大量使用离线技术进行预处理
2. 今天的离线技术可能就是明天的实时技术
3. 统一的理解有助于构建完整的知识逻辑

### 两类方法对比

| 特性 | 离线渲染 | 实时渲染 |
|------|---------|---------|
| 目标品质 | 照片级真实 | 可接受的视觉品质 |
| 计算时间 | 分钟/小时/天 | 16-33ms（60-30fps） |
| 主要技术 | 路径追踪、光子映射 | 光栅化、预计算 |
| 典型应用 | 电影、动画 | 游戏、VR/AR |
