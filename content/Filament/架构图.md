## 渲染管线架构

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                     Filament 渲染管线 (Frame Graph)                          │
└─────────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────────┐
│                              Frame Graph Builder                             │
│                                                                              │

└─────────────────────────────────────────────────────────────────────────────┘
                                    │
                                    ▼
┌─────────────────────────────────────────────────────────────────────────────┐
│                         渲染阶段 (Render Passes)                             │
│                                                                              │
│  ┌────────────────────────────────────────────────────────────────────────┐ │
│  │ 1. Shadow Map Pass (阴影生成)                                           │ │
│  │    │                                                                    │ │
│  │    ├── Directional Light Cascades (级联阴影)                            │ │
│  │    │   ├── Cascade 0 (近处高精度)                                       │ │
│  │    │   ├── Cascade 1                                                    │ │
│  │    │   ├── Cascade 2                                                    │ │
│  │    │   └── Cascade 3 (远处低精度)                                       │ │
│  │    │                                                                    │ │
│  │    ├── Spot Light Shadows                                               │ │
│  │    └── Point Light Shadows (Cubemap)                                    │ │
│  └────────────────────────────────────────────────────────────────────────┘ │
                                    │
                                    ▼
┌─────────────────────────────────────────────────────────────────────────────┐
│  ┌────────────────────────────────────────────────────────────────────────┐ │
│  │ 2. Color Pass (主场景渲染)                                              │ │
│  │    │                                                                    │ │
│  │    ├── Depth Pre-pass (可选)                                            │ │
│  │    │                                                                    │ │

│  │    │                                                                    │ │
│  │    ├── Opaque Geometry (不透明物体)                                     │ │
│  │    │   ├── Standard Materials                                           │ │
│  │    │   ├── Subsurface Materials                                         │ │
│  │    │   └── Cloth Materials                                              │ │

│  │    │                                                                    │ │
│  │    └── Transparent Geometry (透明物体)                                  │ │
│  │        ├── Sorted Back-to-Front                                         │ │
│  │        └── Refraction                                                   │ │
│  └────────────────────────────────────────────────────────────────────────┘ │
                                    │
                                    ▼
┌─────────────────────────────────────────────────────────────────────────────┐
│  ┌────────────────────────────────────────────────────────────────────────┐ │

```

│  │                                                                         ││
│  │ fragment {                                                              ││
│  │     void material(inout MaterialInputs material) {                      ││
│  │         prepareMaterial(material);                                      ││
