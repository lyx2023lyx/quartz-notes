## 一、地形系统概述

### 1.1 什么是地形系统？

```
【通俗理解】
地形系统 = 虚拟世界的"大地"

就像搭积木：
• 普通静态网格体 = 单个积木块
• 地形系统 = 一整块可以任意塑形的"橡皮泥"

地形系统的特点：
┌────────────────────────────────────────────────────────┐
│  ✅ 大面积自然地形                                      │
│     山脉、丘陵、平原、峡谷                              │
│                                                         │
│  ✅ 实时编辑 sculpting                                  │
│     像玩黏土一样推拉地形                                │
│                                                         │
│  ✅ 自动LOD                                             │
│     远处简化，近处精细，自动优化                        │
│                                                         │
│  ✅ 多材质混合                                          │
│     草地、泥土、石头、雪地自然过渡                      │
│                                                         │
│  ✅ 植被系统结合                                        │
│     一键种草、种树                                      │
└────────────────────────────────────────────────────────┘
```

### 1.2 地形 vs 静态网格体

```
什么时候用地形？什么时候用模型？

┌─────────────────┬───────────────────────────────┐
│ 用 地形系统     │ 用 静态网格体                 │
├─────────────────┼───────────────────────────────┤
│ • 大范围自然地形│ • 人工建筑                    │
│ • 山脉、丘陵    │ • 房屋、城墙                  │
│ • 地面、道路    │ • 室内地板                    │
│ • 需要 sculpting│ • 精确几何形状                │
│ • 多材质混合    │ • 复杂建筑结构                │
└─────────────────┴───────────────────────────────┘

实际项目：
┌───────────────────────────────────────────────────┐
│                                                   │
│        ⛰️ 山脉（地形）                            │
│            ↓                                      │
│      ┌──────────┐                                 │
│      │ 🏠 房子   │ ← 静态网格体                   │
│      │  (模型)   │                                 │
│      └────┬─────┘                                 │
│           │                                       │
│  ═════════╧═════════ ← 地面（地形或模型）        │
│                                                   │
└───────────────────────────────────────────────────┘
```

## 二、创建地形

### 2.1 创建新地形

```
步骤：

1. 打开地形模式
   模式工具栏 → 选择【地形】图标

2. 创建新地形
   左侧面板 → 【管理】→ 【创建新地形】

┌─────────────────────────────────────────────┐
│ Create New Terrain 创建新地形               │
├─────────────────────────────────────────────┤
│ 【创建方式】                                 │
│ ○ 新建地形       ← 选这个                   │

│ ○ 从贴图导入                                 │
├─────────────────────────────────────────────┤
│ 【分区大小】                                 │
```

### 2.2 地形尺寸理解

```
【关键概念】

地形由"区块（Section）"组成

┌─────────────────────────────────────────────┐
│                                             │
│  一个地形 = 多个区块拼成                     │
│                                             │
│  ┌────┬────┬────┐                          │
│  │ 1,0│ 1,1│ 1,2│  ← 第1行区块             │
│  ├────┼────┼────┤                          │
│  │ 0,0│ 0,1│ 0,2│  ← 第0行区块             │
│  └────┴────┴────┘                          │
│                                             │
│  每个区块 = 63x63 或 127x127 四边形         │
│  （在创建时选择）                           │
│                                             │
└─────────────────────────────────────────────┘

【推荐设置】
• 初学者：5x5 或 7x7 区块，每区块63x63
• 中等场景：9x9 区块
• 大世界：World Partition 分区加载
```

### 2.3 从高度图导入

```
【什么是高度图？】

高度图 = 灰度图 = 地形的高程数据

┌─────────────────────────────────────────────┐
│  图片像素值 = 地形高度                       │
│                                             │
│  白色 = 高处（山峰）                         │
│  黑色 = 低处（谷底）                         │
│  灰色 = 中间（丘陵）                         │
│                                             │
│  ┌─────────────────┐                        │
│  │  ⬜ ⬜ ⬜ ⬜ ⬜   │  山峰（白）           │
│  │  ⬜ ⬛ ⬛ ⬜ ⬜   │  山谷（黑）           │
│  │  ⬜ ⬛ ⬛ ⬜ ⬜   │                        │
│  │  ⬜ ⬜ ⬜ ⬜ ⬜   │                        │
│  └─────────────────┘                        │
│                                             │
│  常用软件生成高度图：                         │
│  • World Creator                            │
│  • World Machine                            │
│  • Gaea                                     │
│  • Photoshop（手绘）                         │
└─────────────────────────────────────────────┘

【导入步骤】
1. 准备高度图（PNG/TGA，16bit灰度最佳）
2. 地形模式 → 管理 → 从文件导入
3. 选择高度图
4. 设置Z缩放（高度缩放）
5. 导入
```

### 3.4 雕刻工作流程

```
【推荐 sculpting 流程】

第1步：大形（Large Shapes）
├── 大笔刷（1000+）
├── 雕刻山脉、山谷的大轮廓
└── 不要关注细节

第2步：中形（Medium Shapes）
├── 中等笔刷（300-500）
├── 添加山脊、沟壑
└── 定义地形特征

第3步：细节（Details）
├── 小笔刷（50-200）
├── 添加小起伏、岩石
└── 侵蚀工具增加真实感

第4步：抛光（Polish）
├── 平滑工具过渡硬边
├── 平坦工具做平台
└── 检查整体效果

💡 像画画一样：先整体，后局部，再细节
```

## 四、地形材质

### 4.1 为什么需要特殊地形材质？

```
【普通材质的问题】

普通材质：
• 整个地形只能有一种材质
• 要么全是草，要么全是石头

地形材质：
• 可以同时有草地、泥土、石头
• 可以混合过渡
• 可以按高度/坡度自动分配

┌─────────────────────────────────────────────┐
│  普通材质效果：      地形材质效果：          │
│                                             │
│  ┌──────────┐      ┌──────────┐            │
│  │          │      │▓▓▓▓░░▒▒▒│            │
│  │ ▓▓▓▓▓▓▓▓ │      │▓▓░░░░▒▒▒│            │
│  │ ▓▓▓▓▓▓▓▓ │      │░░░░▒▒▒▒▒│            │
│  │ ▓▓▓▓▓▓▓▓ │      │░░▒▒▒▒▒▒▒│            │
│  └──────────┘      └──────────┘            │
│                                             │
│  全是草（单调）     草-泥-石自然过渡        │
└─────────────────────────────────────────────┘
```

### 4.2 创建地形材质

```
【步骤】

1. 创建母材质
   右键 → 材质 → 命名 M_Landscape_Master

2. 打开材质编辑器
   需要特殊节点：LandscapeLayerBlend

3. 设置材质属性
   ┌─────────────────────────────────────────┐
│ Material Domain: Surface                │
│ Blend Mode: Opaque                      │
│ Shading Model: Default Lit              │
│ ☐ Use Material Attributes               │
└─────────────────────────────────────────┘

4. 添加层混合节点
   右键 → Landscape → LandscapeLayerBlend
```

### 4.3 地形材质节点设置

```
【Landscape Layer Blend 节点】

节点结构：

           │         (grass)             │
           └──────────────┬──────────────┘
                          │
                          ▼
           ┌─────────────────────────────┐
           │   Landscape Layer Blend     │
           │  ┌─────┬─────┬─────┬─────┐  │
           │  │Grass│Dirt │Rock │Snow │  │

                    [Break Material
                     Attributes]

          ↓       ↓       ↓       ↓       ↓
     BaseColor Normal Roughness Metallic etc.
```

### 4.4 层混合类型

```
【Landscape Layer Blend 类型】

LB_Weight Blend（权重混合）
• 最常用！
• 层之间可以0-100%混合
• 所有层权重加起来 = 1
• 适合：草地、泥土自然过渡

LB_Alpha Blend（Alpha混合）
• 基于Alpha通道混合
• 需要特殊的权重贴图
• 适合：精确控制过渡

LB_Height Blend（高度混合）
• 基于高度图混合
• 自动处理层之间的遮挡
• 适合：石头突出草地

【推荐设置】
• 大部分层用：Weight Blend
• 需要高度效果的层（如石头）：Height Blend
```

### 5.3 绘制技巧

```
【绘制工作流程】

第1步：基础层
• 选择 Grass 层
• 给整个地形涂满草
• Paint Value = 1.0

第2步：添加变化
• 选择 Dirt 层
• 在低洼处涂泥土
• Paint Value = 0.5-0.8

第3步：增加细节
• 选择 Rock 层
• 在陡坡涂石头
• 可以开启【自动斜坡】检测

第4步：高海拔
• 选择 Snow 层
• 在山顶涂雪
• 可以用高度检测自动分配

【快捷键】
• 左键 = 绘制
• Shift + 左键 = 擦除
• Ctrl + Z = 撤销
```

## 六、高级地形功能

### 6.1 自动材质（Auto-Material）

```
【概念】
让材质自动根据地形属性分配

• 陡坡自动显示石头
• 高处自动显示雪
• 低处自动显示泥土

【实现方式】
在材质中添加 WorldAligned 节点：

Absolute World Position → Component Mask (Z)
         │
         ↓
   [Divide by Height Scale]
         │
         ↓
   [Saturate (0-1)]  ← 高度0-1映射
         │
         ↓
   [Lerp]  ← 根据高度混合材质
    /   \
 Grass   Snow

【Slope Based Blending】
基于坡度的混合：
• 获取地形法线
• 计算坡度角度
• 陡坡 = 石头
• 平坡 = 草地
```

### 6.2 地形样条线（Splines）

```
【用途】
在地形上创建道路、河流、路径

【创建道路步骤】

1. 地形模式 → 【管理】→ 【编辑样条线】

2. 按住 Ctrl + 点击地形添加控制点

3. 调整控制点：
   ┌─────────────────────────────────────────┐
│ 段（Segment）设置：                        │
│ • Width: 道路宽度（如500cm）             │
│ • Side Falloff: 边缘衰减                  │
│ • End Tangent: 切线控制曲线               │
└─────────────────────────────────────────┘

4. 分配材质：
   • 段 → 材质槽 → 选择道路材质

5. 效果：
   地形自动变平，形成道路

【常用设置】
• Raise/Lower Terrain: 抬高/降低地形
• Paint Layer: 自动绘制层（如道路材质）
• Segment Mesh: 放置道路模型
```

### 6.3 地形导入导出

```
【导出高度图】

用途：
• 在其他软件编辑
• 备份地形数据
• 生成小地图

步骤：
地形模式 → 管理 → 导出
选择：Heightmap .png

【导入高度图】

支持格式：

步骤：

4. 导入
```

## 七、地形优化

### 7.1 LOD设置

```
【地形自动LOD】

地形会根据距离自动切换细节：

近距离：            远距离：
┌──────────┐       ┌────┐
│  ⬡⬡⬡⬡⬡   │       │▓▓▓▓│
│  ⬡⬡⬡⬡⬡   │       │▓▓▓▓│
│  ⬡⬡⬡⬡⬡   │       └────┘
└──────────┘       面数减少

【LOD设置】
细节面板 → Landscape → LOD Distribution

LOD Distance:
• LOD0: 0-1000      ← 最近，最精细
• LOD1: 1000-3000   
• LOD2: 3000-6000   
• LOD3: 6000+       ← 最远，最简化
```

### 7.2 性能检查清单

```
✅ 地形优化检查：

□ 区块数量合适？（不要太大）
□ 材质层数合理？（建议4-6层）
□ 贴图分辨率适中？（1024-2048）
□ 使用LOD？（默认开启）
□ 不需要的细节 sculpting 已清除？
□ 层绘制没有过度复杂？

【材质层数建议】
• 简单场景：3-4层
• 标准场景：5-6层
• 复杂场景：7-8层（注意性能）
• 超过8层：考虑优化或拆分材质
```
