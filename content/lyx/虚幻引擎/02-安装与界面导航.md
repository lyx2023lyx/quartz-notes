# 02 安装与界面导航

> 🎯 **本章目标**：成功安装UE5，熟悉主界面各个区域

---

## 一、安装虚幻引擎

### 1.1 准备工作

#### 系统要求

| 配置项 | 最低配置 | 推荐配置 |
|--------|----------|----------|
| **系统** | Windows 10 64位 | Windows 11 64位 |
| **CPU** | 四核处理器 | 十二核以上 |
| **内存** | 8 GB | 32 GB+ |
| **显卡** | GTX 1060 | RTX 3060+ |
| **硬盘** | 15GB 空间 | SSD 100GB+ |

💡 **地编建议**：SSD是必须的！加载大场景机械硬盘会卡死

#### 下载 Epic Games Launcher

1. 访问官网：https://store.epicgames.com/zh-CN/
2. 点击右上角"获取 Epic Games 启动器"
3. 下载并安装

### 1.2 安装 UE5

#### 步骤图解

```
打开 Epic Games Launcher
       │
       ▼
┌────────────────────────────────────────┐
│  左侧菜单 → 选择【虚幻引擎】标签页      │
└────────────────────────────────────────┘
       │
       ▼
┌────────────────────────────────────────┐
│  点击【库】→ 引擎版本旁的【+】按钮      │
└────────────────────────────────────────┘
       │
       ▼
┌────────────────────────────────────────┐
│  选择 【5.7 版本】（或当前最新稳定版）  │
│  勾选 【默认组件】即可                  │
│  （初学者不需要额外勾选）               │
└────────────────────────────────────────┘
       │
       ▼
┌────────────────────────────────────────┐
│  选择安装路径 → 点击安装               │
│  💡 建议装在SSD上                      │
└────────────────────────────────────────┘
```

#### 安装组件说明

| 组件 | 是否勾选 | 说明 |
|------|----------|------|
| **核心组件** | ✅ 必须 | 引擎本体 |
| **初学者内容包** | ✅ 建议 | 示例资源，学习用 |
| **模板和功能包** | ✅ 建议 | 项目模板 |
| **引擎源码** | ❌ 可选 | 不需要，C++开发才用 |
| **目标平台** | 按需 | 只做PC可不勾 |

### 2.3 地编推荐设置

| 选项 | 推荐选择 | 说明 |
|------|----------|------|
| **项目类型** | 游戏 | 最通用的选项 |
| **模板** | 空白 | 从零开始学 |
| **蓝图/C++** | 蓝图 | 地编不需要写代码 |

| **光线追踪** | 可选 | 高端显卡建议开启 |

### 3.2 五大核心区域详解

#### ① 主视口（Viewport）- 你的舞台

这是你最常待的地方，就像摄影师的取景器：

```
┌──────────────────────────────────────────┐
│  ┌┬┬┬┬┬┬┬┬┬┬┬┬┬┬┬┬┬┬┬┬┬┬┬┬┬┬┬┬┬┬┬┐  │
│  ├┴┴┴┴┴┴┴┴┴┴┴┴┴┴┴┴┴┴┴┴┴┴┴┴┴┴┴┴┴┴┴┤  │ ← 透视图/顶视图/侧视图切换
│  │                                  │  │
│  │         3D 场景视图               │  │
│  │                                  │  │
│  │    (这里显示你搭建的场景)          │  │
│  │                                  │  │
│  │                                  │  │
│  │  x - 选中物体的变换控件            │  │
│  │     (移动/旋转/缩放)              │  │
│  │                                  │  │
│  │                                  │  │
│  └──────────────────────────────────┘  │
│     [透视] [光照] [显示] [游戏视图]      │ ← 视图选项
└──────────────────────────────────────────┘
```

**视图模式切换**：
- **Alt + G** = 游戏视图（隐藏编辑器图标）
- **Alt + 2** = 线框模式
- **Alt + 3** = 无光照模式
- **Alt + 4** = 光照模式（常用）
- **Alt + 5** = 细节光照

#### ② 大纲视图（Outliner）- 场景目录

```
┌─────────────────┐
│  🔍 搜索...      │
├─────────────────┤
│  PersistentLevel│ ← 当前关卡
│  ├─ Light Source│ ← 光源
│  ├─ SkyLight    │ ← 天光
│  ├─ PlayerStart │ ← 玩家出生点
│  ├─ Floor       │ ← 地板
│  └─ ...         │
│                 │
│ [+] 添加新对象   │
└─────────────────┘
```

**常用操作**：
- **单击** = 选中物体
- **双击** = 选中并聚焦（F键效果）
- **右键** = 菜单操作
- **拖拽** = 调整层级（创建父子关系）

#### ③ 细节面板（Details）- 属性编辑器

```
┌─────────────────────────┐
│  选中对象: Floor        │
├─────────────────────────┤
│ ▼ Transform 变换        │
│   Location 位置   x y z │
│   Rotation 旋转   x y z │
│   Scale    缩放   x y z │
├─────────────────────────┤
│ ▼ Static Mesh 静态网格   │
│   [选择网格体...]        │
│   [圆柱体]              │
├─────────────────────────┤
│ ▼ Materials 材质        │
│   [0] M_Basic_Floor     │
├─────────────────────────┤
│ ▼ Lighting 光照         │
│   Cast Shadow ☑         │
│   ...                   │
└─────────────────────────┘
```

💡 **重要**：面板右上角可以**锁定**，防止切换选择时变化

#### ④ 内容浏览器（Content Browser）- 资源仓库

```
┌────────────────────────────────────┐
│ 📁 /Game/                          │ ← 当前路径
│ [导入] [新增] [保存全部]            │
├────────────────────────────────────┤
│ ┌────┐ ┌────┐ ┌────┐ ┌────┐       │
│ │📁  │ │📦  │ │🔵  │ │💡  │       │
│ │文件│ │网格│ │材质│ │蓝图│       │
│ └────┘ └────┘ └────┘ └────┘       │
│         ...                        │
│                                    │
│  选中资源 → 右键拖入场景           │
└────────────────────────────────────┘
```

**资源类型图标**：
- 📁 = 文件夹
- 📦 = 静态网格体（模型）
- 🔵 = 材质/材质实例
- 🔴 = 贴图
- 💡 = 蓝图
- 🗺️ = 关卡

#### ⑤ 工具栏 - 快捷操作

```
┌────────────────────────────────────────────────────────────┐
│ [💾] [↩] [↪] │ [选择] [地形] [植被] [笔刷] │ [▶播放] │ ⚙️  │
└────────────────────────────────────────────────────────────┘
         │        │      │      │      │
         │        │      │      │      └─ 模式切换
         │        │      │      └─ 植被绘制模式
         │        │      └─ 地形编辑模式
         │        └─ 默认选择模式
         └─ 文件操作
```

## 四、窗口布局自定义

### 4.1 保存自定义布局

```
窗口 → 加载布局 → 保存布局 → 命名"地编工作区"
```

```

│  大纲视图    │          主视口（大）            │  细节面板  │
│  (窄一点)    │                                  │  (宽一点)  │

```
