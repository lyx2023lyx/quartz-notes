## 二、Directional Light 方向光

### 2.1 什么是方向光？

```
【通俗理解】
方向光 = 太阳光（或月光）

特点：
• 光线平行（因为太阳太远）
• 没有具体位置，只有方向
• 照亮整个场景
• 产生平行阴影

┌─────────────────────────────────────────────┐
│                                             │
│    \\  \\  \\  \\  \\  \\  \\  \\  \\         │
│     \\  \\  \\  \\  \\  \\  \\  \\  \\        │
│      \\  \\  \\  \\  \\  \\  \\  \\  \\       │
│       \\  \\  \\  \\  \\  \\  \\  \\  \\      │
│        ═══════════════════════             │
│              地面                           │
│                                             │
│  所有光线平行，方向一致                      │
│  就像太阳光照到地球                          │
└─────────────────────────────────────────────┘
```

### 2.2 方向光属性

```
选中 DirectionalLight → 细节面板

┌─────────────────────────────────────────────┐
│ Light                                         │
├─────────────────────────────────────────────┤
│ Intensity 强度                               │
│   •  lux（勒克斯）物理单位                   │
│   • 室外晴天：10000-100000 lux              │
│   • 室内灯光：100-1000 lux                  │
│   • 月光：0.1-1 lux                         │
├─────────────────────────────────────────────┤
│ Light Color 光色                             │
│   • 正午：白色/淡黄（6500K）                │
│   • 日落：橙红（3000K）                     │
│   • 阴天：蓝灰（8000K）                     │
├─────────────────────────────────────────────┤
│ Temperature 色温 ☑                           │
│   • 勾选后按色温调整                         │
│   • 数值越低越暖（红），越高越冷（蓝）      │
├─────────────────────────────────────────────┤
│ Source Angle 光源角度                        │
│   • 影响阴影虚化程度                         │
│   • 太阳：0.5度（锐利阴影）                 │
│   • 大光源：增大角度（柔和阴影）            │
├─────────────────────────────────────────────┤
│ Cast Shadows 投射阴影 ☑                      │
│   • 必须勾选！否则没有阴影                  │
├─────────────────────────────────────────────┤
│ Atmosphere Sun 大气太阳光 ☑                  │
│   • 关联SkyAtmosphere，旋转方向光=改变太阳位置│
└─────────────────────────────────────────────┘
```

### 2.3 操作技巧

```
【改变太阳方向】
• 选中方向光
• 旋转它（E键旋转模式）
• 观察阴影方向变化

【太阳位置可视化】
• 方向光绑定 SkyAtmosphere
• 旋转方向光，天空中的太阳会跟着动
• 同时影响时间氛围（正午/黄昏/夜晚）

【快捷键】
• Ctrl + L + 鼠标移动 = 快速改变光照方向
  （类似Maya的灯光操作）
```

## 三、Point Light 点光源

### 3.1 点光源原理

```
【通俗理解】
点光源 = 灯泡、火把、蜡烛

特点：
• 从一个点向四周发光
• 光线随距离衰减
• 强度随距离平方衰减（真实物理）

┌─────────────────────────────────────────────┐
│                                             │
│              ● 点光源                       │
│             /|\                             │
│            / | \                            │
│           /  |  \                           │
│          /   |   \                          │
│         /    |    \                         │
│        /     |     \                        │
│       ═══════╧═══════                      │
│              地面                           │
│                                             │
│  光线360度散射                              │
│  越近越亮，越远越暗                         │
└─────────────────────────────────────────────┘
```

### 3.2 点光源属性

```
┌─────────────────────────────────────────────┐
│ Point Light 属性                             │
├─────────────────────────────────────────────┤
│ Intensity 强度                               │
│   • 单位：流明（lm）或坎德拉（cd）          │
│   • 灯泡：100-1000 lm                       │
│   • 火把：500-2000 lm                       │
│   • 路灯：1000-5000 lm                      │
├─────────────────────────────────────────────┤
│ Attenuation Radius 衰减半径                  │
│   • 光能照到的最远距离                       │
│   • 黄色线球显示范围                         │
│   • 设置合适大小（不要太大浪费性能）        │
├─────────────────────────────────────────────┤
│ Falloff Exponent 衰减指数                    │
│   • 默认2 = 物理正确                         │
│   • 值越大，边缘衰减越快                     │
├─────────────────────────────────────────────┤
│ Source Radius 光源半径                       │
│   • 影响阴影虚化                             │
│   • 大光源=柔和阴影，小光源=锐利阴影        │
├─────────────────────────────────────────────┤
│ Cast Shadows 投射阴影 ☑                      │
│   • 动态物体投阴影                           │
│   • 静态物体建议烘焙阴影                     │
└─────────────────────────────────────────────┘
```

### 3.3 使用技巧

```
【室内布光示例】

房间布光：
┌───────────────────────────────────────┐
│  💡                                    │
│  天花板灯（点光源）                    │
│        │                               │
│        ▼                               │
│  ┌─────────────┐                       │
│  │             │  🕯️                   │
│  │    房间     │  桌子上的蜡烛         │
│  │             │  （暖色小半径）       │
│  │      🚪    │                       │
│  └─────────────┘                       │
│                                        │
│  窗户透进的Directional Light           │
└───────────────────────────────────────┘

【性能优化】
• 控制Attenuation Radius不要过大
• 重叠的光源不要超过3-4个
• 静态物体用烘焙代替实时光
```

## 四、Spot Light 聚光灯

### 4.1 聚光灯原理

```
【通俗理解】
聚光灯 = 手电筒、舞台灯、车灯、探照灯

特点：
• 有明确方向
• 锥形照射范围
• 内锥：最亮区域
• 外锥：衰减过渡区

┌─────────────────────────────────────────────┐
│                                             │
│              ● 聚光灯                       │
│             /\                              │
│            /  \  ← 外锥（半影）             │
│           /    \                            │
│          /  __  \                           │
│         /  /  \  \  ← 内锥（本影）          │
│        /  /    \  \                         │
│       ══╧══════╧═══                        │
│              地面                           │
│                                             │
│  光线集中在锥形范围内                        │
│  内圈亮，外圈渐暗                           │
└─────────────────────────────────────────────┘
```

### 4.2 聚光灯属性

```
┌─────────────────────────────────────────────┐
│ Spot Light 特有属性                          │
├─────────────────────────────────────────────┤
│ Inner Cone Angle 内锥角度                    │
│   • 光线最亮的中心区域                       │
│   • 范围：0-90度                             │
│   • 手电筒：10-20度                         │
│   • 舞台灯：30-45度                         │
├─────────────────────────────────────────────┤
│ Outer Cone Angle 外锥角度                    │
│   • 光线衰减到零的边界                       │
│   • 必须 > 内锥角度                         │
│   • 决定光锥的柔和度                         │
├─────────────────────────────────────────────┤
│ 其他属性和Point Light相同                    │
│   • Intensity、Color、Attenuation等         │
└─────────────────────────────────────────────┘
```

### 4.3 应用场景

```
【聚光灯使用场景】

1. 手电筒
   Inner: 15°  Outer: 30°  Intensity: 5000

2. 路灯
   Inner: 30°  Outer: 60°  Intensity: 3000
   需要投射阴影（地面光影）

3. 舞台重点照明
   Inner: 20°  Outer: 40°  Intensity: 10000
   暖色（3000K）

4. 探照灯/搜索灯
   Inner: 5°   Outer: 15°  Intensity: 50000
   锐利光束效果

【技巧：光柱效果】
• 添加体积雾（Volumetric Fog）
• 聚光灯在雾中会显示光束
• 营造丁达尔效应
```

## 五、Sky Light 天空光

### 5.1 为什么需要天空光？

```
【问题】
只用DirectionalLight的问题：
• 阴影处全黑（不真实）
• 没有天空的颜色反射

┌─────────────────────────────────────────────┐
│                                             │
│        ☀️ DirectionalLight                 │
│             ↓                               │
│        ┌─────────┐                          │
│        │  照亮   │                          │
│        │   ↓     │                          │
│        ═══════════                          │
│        ▓▓▓ 阴影处全黑 ▓▓▓                  │
│                                             │
│  现实中阴影处应该被天空反射光照亮            │
└─────────────────────────────────────────────┘

【天空光的作用】
模拟天空/环境对场景的间接光照：
• 阴影处不是全黑
• 获得天空的颜色
• 整体更真实自然
```

## 七、光照移动性（Mobility）

### 7.1 三种移动性

```
┌─────────────────────────────────────────────────────────┐
│                 光照移动性设置                           │
├───────────────┬──────────────────┬─────────────────────┤
│ 移动性        │ 特点             │ 适用场景            │
├───────────────┼──────────────────┼─────────────────────┤
│ Static        │ 静态，烘焙光照   │ 不动的一切          │
│ 静态          │ 最高性能         │ 墙、地面、建筑      │
│               │ 最高质量         │ 运行时不能动        │
├───────────────┼──────────────────┼─────────────────────┤
│ Stationary    │ 半静态           │ 会开关/变色的灯     │
│ 固定          │ 烘焙+动态混合    │ 路灯、室内灯        │
│               │ 可以改变亮度和色 │ 位置不能动          │
├───────────────┼──────────────────┼─────────────────────┤
│ Movable       │ 完全动态         │ 移动的光源          │
│ 可移动        │ 实时计算         │ 手电筒、火把        │
│               │ 性能开销大       │ 随玩家移动的光      │
└───────────────┴──────────────────┴─────────────────────┘
```

### 7.2 如何选择

```
【决策流程】

这个光源会移动位置吗？
        │
        ├── 是 → Movable 可移动
        │         （手电筒、车灯）
        │
        └── 否 → 位置固定
                  │
                  ├── 需要改颜色/亮度吗？
                  │       │
                  │       ├── 是 → Stationary 固定
                  │       │         （可开关的灯）
                  │       │
                  │       └── 否 → Static 静态
                  │                 （太阳光）

【地编建议】
• 太阳光：Static（烘焙）或 Stationary
• 室内灯：Stationary
• 玩家手电筒：Movable
• 特效光：Movable
```

### 7.3 混合使用示例

```
【复杂场景布光】

户外场景：
┌─────────────────────────────────────────────┐
│                                             │
│         ☀️ Directional Light               │
│            [Static] 静态烘焙                │
│                                             │
│  🏠 房子                    🚗 车辆        │
│     ↓                           ↓           │
│  [Static Light]            [Stationary]     │
│  门窗透出的光              车灯（可开关）   │
│                                             │
│  玩家手电筒 💡                             │
│  [Movable] 跟随玩家移动                     │
│                                             │
└─────────────────────────────────────────────┘

性能平衡：
• 大部分静态光源烘焙
• 少量动态光源实时计算
• 最佳质量和性能平衡
```

## 八、光照烘焙（Build Lighting）

### 8.1 什么是烘焙？

```
【通俗理解】
烘焙 = 预计算光照

实时计算：
• 每帧都算光影
• 性能开销大
• 适合动态物体

烘焙：
• 提前算好光影，存成贴图
• 运行时直接读取
• 性能极好，质量更高
• 只有静态物体能烘焙

┌─────────────────────────────────────────────┐
│                                             │
│  烘焙前：                                    │
│  ┌─────┐   光线计算中...                     │
│  │ 墙  │ ← 每帧计算光影                     │
│  └─────┘   性能消耗大                        │
│                                             │
│  烘焙后：                                    │
│  ┌─────┐   光照贴图                         │
│  │▓▓▓▓▓│ ← 直接读取贴图                    │
```

### 8.3 烘焙问题排查

```
【常见问题】

❌ "光照需要重建" 红色提示
    解决：构建 → 构建光照

❌ 烘焙后出现黑斑/脏斑
    原因：UV重叠或光照贴图分辨率太低
    解决：检查模型UV2（光照UV）

❌ 静态物体没有阴影
    原因：光源没开Cast Shadow，或物体设置问题
    解决：检查光源和物体的光照设置

❌ 烘焙时间太长
    解决：降低Lightmass设置，或减少静态光源数量

【Lightmass设置】
世界场景设置 → Lightmass
• Static Lighting Level Scale: 1（越小越精细）
• Num Indirect Lighting Bounces: 3（间接光反弹次数）
• Indirect Lighting Quality: 1（质量）
```
