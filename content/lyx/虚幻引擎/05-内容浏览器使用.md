## 一、内容浏览器概述

### 1.1 什么是内容浏览器？

**通俗比喻**：内容浏览器是你的**资源仓库+工具箱**

```
就像摄影师的设备间：
├── 📦 相机设备区（模型资源）
├── 🎨 调色工具区（材质贴图）
├── 💡 灯光设备区（灯光预设）
└── 📁 项目文件夹（按项目分类）

在UE中：
├── 存放所有游戏资源
├── 管理资源分类
├── 导入外部资源
└── 快速拖拽使用
```

## 三、资源导入

### 3.1 导入流程

```
┌──────────────────────────────────────────────────────┐
│                   资源导入流程                        │
├──────────────────────────────────────────────────────┤
│                                                      │
│  1. 准备资源                                          │
│     ├── 模型：FBX格式（推荐）                         │
│     ├── 贴图：PNG/JPG/TGA                            │
│     └── 命名规范：T_Wood_D.png（T_前缀+名字+类型）    │
│                                                      │
│         ↓                                            │
│                                                      │
│  2. 导入到UE                                         │
│     ├── 方法一：内容浏览器 → 右键 → 导入到...       │
│     ├── 方法二：直接拖拽文件到内容浏览器            │
│     └── 方法三：导入按钮                             │
│                                                      │
│         ↓                                            │
│                                                      │
│  3. 处理导入选项                                    │
│     └── 根据资源类型调整参数                        │
│                                                      │
│         ↓                                            │
│                                                      │
│  4. 保存                                            │
│     └── Ctrl+S 保存资源                             │
│                                                      │
└──────────────────────────────────────────────────────┘
```

### 3.2 FBX模型导入设置

```
导入模型时的选项面板：

┌─────────────────────────────────────────────┐
│ FBX 导入选项                                 │
├─────────────────────────────────────────────┤
│ 【网格体】                                   │

├─────────────────────────────────────────────┤
│ 【材质】                                     │
│   ☑ 导入纹理       ← 同时导入贴图           │
│   ☑ 创建材质       ← 自动创建基础材质       │
│   材质导入方法：创建新材质                    │
├─────────────────────────────────────────────┤
│ 【变换】                                     │

💡 初学者建议：保持默认即可，有问题再调整
```

### 3.3 贴图导入设置

```
┌─────────────────────────────────────────────┐
│ 纹理 导入选项                                │
├─────────────────────────────────────────────┤
│ 【纹理类型】                                 │
│   • 颜色/漫反射 (Default)  ← 普通贴图      │
│   • 法线贴图 (Normalmap)   ← 凹凸细节      │
│   • 灰度/蒙版 (Grayscale)  ← 黑白遮罩      │
│   • 蒙版 (Mask)            ← Alpha通道     │
├─────────────────────────────────────────────┤
│ 【压缩设置】                                 │
│   • 默认 (Default DX11/12)                 │
│   • 法线贴图 (Normalmap)                   │
│   • 灰度 (Grayscale)                       │
│   • 用户界面2D (UI)                        │
├─────────────────────────────────────────────┤
│ 【高级】                                     │
│   贴图组：World(地编用)/Character(角色用)   │
│   无透明通道：根据贴图决定                   │
└─────────────────────────────────────────────┘

贴图类型说明：
• _D / _BaseColor = 基础颜色（漫反射）
• _N / _Normal    = 法线贴图（凹凸）
• _M / _Metallic  = 金属度（黑=非金属，白=金属）
• _R / _Roughness = 粗糙度（黑=光滑，白=粗糙）
• _AO / _Occlusion = 环境光遮蔽（阴影细节）
```

### 3.4 导入常见问题

```
┌──────────────────────────────────────────────────────┐
│                   导入问题排查                        │
├──────────────────────────────────────────────────────┤
│                                                      │
│ ❌ 模型导入后很小/很大                                │
│    解决：检查建模软件单位设置，或在导入时调整缩放     │
│                                                      │
│ ❌ 贴图导入后显示不正常（很亮/很暗）                  │
│    解决：检查贴图类型设置（sRGB选项）                │
│                                                      │
│ ❌ 法线贴图显示错误（凹凸反了）                       │
│    解决：翻转绿色通道（Flip Green Channel）          │
│                                                      │
│ ❌ 透明贴图（PNG）不透明                              │
│    解决：确保导入设置中"无透明通道"未勾选            │
│                                                      │
│ ❌ 模型丢失贴图（紫色）                               │
│    解决：重新导入并勾选"导入纹理"，或手动指定贴图    │
│                                                      │
└──────────────────────────────────────────────────────┘
```

```

│   ├── 📁 Maps/              ← 关卡

│   │   └── L_Test.umap
│   │
│   ├── 📁 Environment/       ← 环境

│   │   ├── 📁 Foliage/       ← 植被
│   │   └── 📁 Water/         ← 水体
│   │
│   ├── 📁 Architecture/      ← 建筑
│   │   ├── 📁 Buildings/     ← 房屋
│   │   └── 📁 Structures/    ← 结构
│   │
│   └── 📁 Props/             ← 道具

    ├── 📁 Megascans/         ← Quixel资源

### 4.2 资源命名规范

```
前缀规范（必须遵守！）：

类型          前缀          示例
─────────────────────────────────
关卡          L_            L_Forest_01
静态网格体    SM_           SM_Tree_Pine
材质          M_            M_Wood_Oak
材质实例      MI_           MI_Wood_Red
纹理/贴图     T_            T_Wood_D
蓝图          BP_           BP_Door_Slide
粒子系统      P_            P_Fire_Camp
后期处理      PP_           PP_Daytime

完整示例：
✅ SM_Chair_Wooden_01    ← 好
✅ T_Brick_Red_D         ← 好（颜色+类型）
❌ chair                 ← 不好（无前缀）
❌ MyTexture              ← 不好（无意义命名）
```

### 4.3 资源搜索技巧

```
内容浏览器搜索框：

【基础搜索】
输入关键词 → 显示匹配结果

【高级搜索语法】
Type=StaticMesh         ← 只显示静态网格体
Name=SM_*               ← 显示SM开头的资源
Tag=Foliage             ← 显示有Foliage标签的

【组合搜索】
SM_Tree Type=StaticMesh  ← SM_Tree开头的静态网格体

【快捷筛选】
右下角漏斗图标 → 按类型筛选
```

## 五、资源使用操作

### 5.1 拖拽使用

```
┌──────────────────────────────────────────────────────┐
│                   资源使用方式                        │
├──────────────────────────────────────────────────────┤
│                                                      │
│  【静态网格体】                                       │
│   内容浏览器 → 拖到视口 → 自动创建Actor              │
│                                                      │
│  【材质】                                             │
│   拖到模型上 = 赋予材质                              │
│   拖到细节面板材质槽 = 赋予材质                      │
│                                                      │
│  【贴图】                                             │
│   拖到材质编辑器 = 作为采样节点                      │
│   不能直接拖到场景！                                 │
│                                                      │
│  【蓝图】                                             │
│   拖到视口 = 创建蓝图实例                            │
│                                                      │
│  【关卡】                                             │
│   双击 = 打开该关卡                                  │
│   拖到当前关卡 = 作为子关卡                          │
│                                                      │
└──────────────────────────────────────────────────────┘
```

### 5.2 资源引用查看

```
右键资源 → 引用查看器

功能：
• 查看谁引用了这个资源
• 查看这个资源引用了谁
• 发现 orphaned 资源（没被引用的）

用途：
• 删除资源前检查是否被使用
• 优化时找出无用资源
• 理解资源依赖关系
```

### 5.3 资源迁移

```
场景1：把资源从项目A移到项目B

步骤：
1. 内容浏览器 → 右键资源 → 迁移
2. 选择目标项目文件夹
3. 自动复制该资源及其依赖

场景2：在项目内移动文件夹

直接拖拽即可，引用会自动更新

⚠️ 注意：不要在Windows里直接移动！
      一定要在内容浏览器里操作
```

## 六、初学者内容包

### 6.1 什么是初学者内容？

```
安装UE时勾选的【初学者内容】，包含：

├── 📁 Architecture/          ← 建筑部件
│   ├── SM_Floor_...          ← 各种地板
│   ├── SM_Wall_...           ← 各种墙
│   └── SM_DoorFrame_...      ← 门框
│
├── 📁 Props/                 ← 道具
│   ├── SM_Chair              ← 椅子
│   ├── SM_Table              ← 桌子
│   ├── SM_Lamp_...           ← 灯具
│   └── ...
│
├── 📁 Materials/             ← 基础材质
│   ├── M_Basic_Floor         ← 地板材质
│   ├── M_Basic_Wall          ← 墙面材质
│   ├── M_Glass               ← 玻璃

### 6.2 学习使用建议

```
【第一阶段：纯学习】
• 把初学者内容当"积木"玩
• 熟悉各种资源的用法
• 不用担心版权

【第二阶段：开始原创】
• 开始导入自己的资源
• 创建自己的材质
• 初学者内容作为参考

【第三阶段：项目实战】
• 新项目可以不勾选初学者内容
• 或迁移需要的资源到自己的文件夹
```

## 🎮 练习任务

1. 新建文件夹结构：Maps、Meshes、Materials、Textures
2. 导入一个FBX模型（可以用Blender导出一个简单的立方体）
3. 导入几张贴图（搜索免费纹理网站如 textures.com）
4. 用初学者内容的资源搭建一个小场景
5. 按照命名规范重命名你的资源
