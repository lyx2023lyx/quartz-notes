# 07 材质系统入门

> 🎯 **本章目标**：理解材质原理，学会创建和编辑基础材质

---

## 一、什么是材质？

### 1.1 通俗解释

**材质 = 物体的"皮肤"**

```
想象现实世界：

同一把椅子，不同材质：
┌─────────────┐  ┌─────────────┐  ┌─────────────┐
│   木椅子    │  │   铁椅子    │  │   塑料椅子  │
├─────────────┤  ├─────────────┤  ├─────────────┤
│ • 棕色      │  │ • 银灰色    │  │ • 五颜六色  │
│ • 有木纹    │  │ • 反光      │  │ • 哑光      │
│ • 不反光    │  │ • 冰冷感    │  │ • 光滑      │
│ • 粗糙      │  │ • 光滑      │  │ • 轻飘感    │
└─────────────┘  └─────────────┘  └─────────────┘

在UE中，材质决定：
├── 颜色（Color）
├── 反光度（Metallic/Roughness）
├── 纹理（Texture）
└── 特殊效果（发光、透明等）
```

### 1.2 PBR 工作流

```
PBR = Physically Based Rendering = 基于物理的渲染

简单说：材质参数符合真实物理规律

地编最常用的 PBR 工作流：

【Metallic/Roughness 工作流】
                    │
        ┌───────────┼───────────┐
        │           │           │
        ▼           ▼           ▼
   ┌─────────┐ ┌─────────┐ ┌─────────┐
   │BaseColor│ │Metallic │ │Roughness│
   │基础颜色 │ │金属度   │ │粗糙度   │
   └─────────┘ └─────────┘ └─────────┘
        │           │           │
        │    ┌──────┴──────┐    │
        │    │             │    │
        └────┤    材质      ├────┘
             │   (输出)    │
             └─────────────┘

三个核心参数：
• BaseColor = 物体本身的颜色（木头是棕色，铁是灰色）
• Metallic = 是否是金属（0=塑料/木头，1=金/银/铁）
• Roughness = 表面粗糙度（0=镜子，1=磨砂）
```

### 2.3 核心概念：节点

```
【节点 = 材质的功能模块】

每个节点做一件事，用线连起来组成材质逻辑：

输入节点                          输出节点
┌─────────────┐                   ┌─────────────┐
│  贴图样本   │                   │   结果      │
│ Texture     │───┐           ┌──→│  Result    │
│ Sample      │   │           │   └─────────────┘
└─────────────┘   │   ┌───┐   │        ↑
                  └──→│ × │───┘   最终输出到渲染
┌─────────────┐   ┌──→│   │              
│  颜色常量   │   │   └───┘               
│ Constant3   │───┘   数学节点           
└─────────────┘                       

常用节点类型：
• 常量节点 = 固定数值（如颜色0.5,0.5,0.5）
• 参数节点 = 可调数值（材质实例用）
• 贴图节点 = 导入的图片
• 数学节点 = 加减乘除（如Multiply混合）
• 函数节点 = 复杂功能（如旋转UV）
```

## 三、创建第一个材质

### 3.1 基础颜色材质

```
【最简单的材质】纯色

步骤：
1. 创建材质 M_Solid_Red
2. 双击打开材质编辑器
3. 右键搜索 "Constant3Vector" → 颜色常量
4. 双击节点设置颜色为红色
5. 连线到 "Base Color"
6. 点击【应用】→【保存】

图表：
┌─────────────────────┐
│  Constant3Vector    │
│  [R:1 G:0 B:0]      │────→ Base Color
│  （红色）           │
└─────────────────────┘
```

### 3.2 使用贴图的材质

```
【带纹理的材质】木纹

步骤：
1. 导入木纹贴图 T_Wood_D.png
2. 创建材质 M_Wood
3. 在图表中右键 → Texture Sample
4. 选择 T_Wood_D
5. 连线 RGB → Base Color

图表：
┌─────────────────────┐
│  Texture Sample     │
│  T_Wood_D           │────→ Base Color
│  （木纹贴图）        │
└─────────────────────┘

💡 贴图后缀含义：
• _D / _BaseColor = 基础颜色
• _N / _Normal    = 法线（凹凸）
• _M / _Metallic  = 金属度
• _R / _Roughness = 粗糙度
• _AO             = 环境光遮蔽
```

需要的贴图：

## 四、材质属性详解

### 4.1 材质细节面板

```
【关键属性设置】

┌─────────────────────────────────────────────┐
│ Physical Material 物理材质                   │
├─────────────────────────────────────────────┤
│ 【Material】                                 │
│   Material Domain: Surface                  │
│     • Surface = 表面材质（99%用这个）        │

│     • Post Process = 后期处理                │
│                                              │
│   Blend Mode: Opaque                        │

│                                              │

│ 【Translucency】（只有Blend Mode选半透明时） │

└─────────────────────────────────────────────┘
```

├─────────────┼──────────────────┼──────────────────────┤

## 五、常用材质节点

### 5.1 基础节点

```
【Texture Sample】贴图采样
• 用途：读取贴图
• 输出：RGB颜色、R/G/B单通道、Alpha
• 设置：选择贴图、采样器类型

【Constant】常量
• Constant = 单值（0-1）
• Constant2Vector = 双值（如UV坐标）
• Constant3Vector = 颜色（RGB）
• Constant4Vector = 颜色+Alpha（RGBA）

【Multiply】乘法
• 用途：混合两个值
• 例子：贴图 × 颜色 = 染色贴图
• 例子：BaseColor × AO = 有阴影的颜色

【Add】加法
• 用途：叠加两个值
• 例子：颜色 + 发光 = 自发光

【Lerp】线性插值
• 用途：两个值之间混合
• Alpha = 0时输出A，Alpha = 1时输出B
• 例子：干地面 过渡到 湿地
```

### 5.2 UV相关节点

```
【UV基础】
UV = 贴图坐标，告诉材质怎么把2D贴图贴到3D模型上

┌───────────────────────────┐
│     UV坐标系（0-1空间）    │
│                           │
│   (0,1)        (1,1)      │
│     ┌──────────┐          │
│     │          │          │
│     │  贴图    │          │
│     │          │          │
│     └──────────┘          │
│   (0,0)        (1,0)      │
│                           │
└───────────────────────────┘

常用UV节点：
• TextureCoordinate = 基础UV
• Panner = UV平移（流水、飘云）
• Rotator = UV旋转
• FlipBook = 序列帧动画

UV缩放示例：
TextureCoordinate → Multiply(×2) → 贴图
= 贴图重复2次（平铺）
```

### 5.3 世界对齐节点

```
【World Aligned Texture】世界对齐贴图

问题：地形斜坡贴图会拉伸
解决：让贴图沿着世界坐标对齐，不是模型UV

用法：
World Aligned Texture → Base Color
                       ↑
                  选择贴图

效果：
┌────────────────────────────┐
│  地形：                     │
│        ╱╲                  │
│       ╱  ╲                 │
│      ╱    ╲                │
│                             │
│  普通UV：贴图跟着斜坡拉伸   │
│  世界对齐：贴图保持水平     │
└────────────────────────────┘

地编必备：地形材质用它避免拉伸！
```

## 六、材质实例

### 6.1 什么是材质实例？

```
【概念】
母材质（Master Material）= 基础配方
材质实例（Material Instance）= 基于配方的变种

类比：
母材质 = 基础奶茶配方
├── MI_奶茶_珍珠（+珍珠）
├── MI_奶茶_椰果（+椰果）
└── MI_奶茶_布丁（+布丁）

好处：
• 改母材质，所有实例都变
• 实例参数可调，不用重新编译
• 性能优化（实例渲染更快）
```

### 6.2 创建材质实例

```
步骤：
1. 创建母材质 M_Master_Wall
2. 设置需要暴露的参数（右键 → 转换为参数）
3. 保存母材质
4. 右键母材质 → 创建材质实例
5. 命名 MI_Wall_Red、MI_Wall_Blue

参数类型：
┌────────────────────────────────┐
│ Scalar Parameter 标量参数      │
│   • 数字（0-1）                │
│   • 如：Roughness值            │
├────────────────────────────────┤
│ Vector Parameter 向量参数      │
│   • 颜色（RGB）                │
│   • 如：BaseColor颜色          │
├────────────────────────────────┤
│ Texture Parameter 贴图参数     │
│   • 可更换的贴图               │
│   • 如：Diffuse贴图            │
└────────────────────────────────┘
```

### 6.3 材质实例编辑器

```
材质实例界面（简化版）：

┌─────────────────────────────────────────────┐
│ Parent: M_Master_Wall                       │
├─────────────────────────────────────────────┤
│ General                                     │
│   ☑ Override Base Color                    │
│   Base Color: [红色选择器] ⬜              │
│                                             │
│   ☑ Override Roughness                     │
│   Roughness: [0.5] ────────⬜             │
│                                             │
│   ☑ Override Diffuse Texture               │
│   Diffuse: [T_Wall_Brick] ⬜               │
├─────────────────────────────────────────────┤
│ Preview                                     │
│   [3D预览球体]                              │
└─────────────────────────────────────────────┘

勾选 ☑ = 覆盖母材质的默认值
```

## 七、地编常用材质类型

### 7.1 地形混合材质

```
【地形需要混合多种材质】

地形材质特点：
• 草地、泥土、石头混合
• 用Weight Map控制混合
• 每层可以独立调整

节点结构：
┌───────────────────────────────────────┐
│ Layer 1: 草地                          │
│ Layer 2: 泥土                          │
│ Layer 3: 石头                          │
│         ↓                             │
│    [Landscape Layer Blend]             │
│         ↓                             │
│     混合输出 → Base Color              │
└───────────────────────────────────────┘

每层包含：
• Albedo（颜色）
• Normal（法线）
• Roughness（粗糙度）
```

### 7.2 顶点绘制材质

```
【Vertex Paint = 顶点绘制】

用途：在模型上手动涂抹不同材质

例子：
┌───────────────────────────────┐
│  墙面：                        │
│  ┌─────────────────┐          │
│  │ 砖 ████ 石头    │          │
│  │ 砖 ████ 石头    │          │
│  │ 砖 ████ 石头    │          │
│  └─────────────────┘          │
│                                │
│  在编辑器里用笔刷涂抹：         │
│  • 红色通道 = 砖               │
│  • 绿色通道 = 石头             │
│  • 蓝色通道 = 苔藓             │
└───────────────────────────────┘

材质节点：Vertex Color → Lerp（混合多个层）
```

### 7.3 视差材质（Parallax）

```
【假深度效果】

效果：看起来有凹凸，但模型是平的

对比：
• Normal贴图 = 改变光照，看起来凹凸
• Parallax = 真的"移动"贴图，更有立体感

用途：
• 砖缝深度
• 地板缝隙
• 墙面细节

注意：
• 性能开销比Normal大
• 侧面看可能穿帮
• 适合第一人称视角
```

## 八、材质优化

### 8.1 性能原则

```
┌───────────────────────────────────────────────────────┐
│                  材质性能优化                          │
├───────────────────────────────────────────────────────┤
│                                                        │
│ 【贴图尺寸】                                           │
│ • 尽量用2的幂次方（512、1024、2048）                  │
│ • 远处物体用小贴图（512）                              │
│ • 近处重要物体用大贴图（2048）                         │
│ • 避免用奇数尺寸（如1000×1000）                       │
│                                                        │
│ 【贴图压缩】                                           │
│ • 颜色贴图：DXT1/BC1（压缩高）                         │
│ • 法线贴图：DXT5/BC5（保留细节）                       │
│ • UI贴图：不压缩（保持锐利）                           │
│                                                        │
│ 【材质复杂度】                                         │
│ • 节点越少越好                                         │
│ • 少用复杂数学计算                                     │
│ • 半透明材质尽量少用（性能杀手）                       │
│                                                        │
│ 【Draw Call】                                          │
│ • 相同材质合并（Static Mesh合并）                      │
│ • 使用材质实例而不是多个母材质                         │
│                                                        │
└───────────────────────────────────────────────────────┘
```

### 8.2 检查清单

```
✅ 材质发布前检查：

□ 贴图已经压缩？
□ 不需要的贴图已经移除？
□ Blend Mode选择正确？
□ Two Sided只在需要时开启？
□ 半透明材质数量合理？
□ 材质实例参数命名清晰？
□ 预览中效果正确？
```

|------|--------|------|
| 应用 | 工具栏按钮 | 编译材质 |
| 保存 | Ctrl+S | 保存材质 |
| 创建节点 | 右键/空格 | 添加节点 |
| 连线 | 拖拽 | 连接节点 |
| 断线 | Alt+点击 | 断开连接 |

## 🎮 练习任务

1. 创建一个纯红色的基础材质
2. 导入一张木纹贴图，创建木纹材质
3. 创建一个完整的PBR金属材质（用4张贴图）
4. 创建母材质+3个不同颜色的材质实例
5. 尝试创建一个半透明玻璃材质

---

> 🐸 **青蛙提示**：材质是地编的"调色板"，好地编都是半个美术。记住PBR三件套：颜色（BaseColor）、金属度（Metallic）、粗糙度（Roughness），90%的材质都靠它们！
