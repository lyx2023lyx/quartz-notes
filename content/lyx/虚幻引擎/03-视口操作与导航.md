# 03 视口操作与导航

> 🎯 **本章目标**：掌握在3D场景中自由移动、观察、选择物体的技能

---

## 一、视口基础概念

### 1.1 什么是视口？

**比喻**：视口就是你的"摄影机取景器"

就像拍电影时导演通过监视器看画面，你在UE里通过视口"看"你的3D场景：
- 可以在场景中**到处飞**（导航）
- 可以**拉近看细节**或**拉远看全景**
- 可以切换不同的**观察角度**

### 1.2 视口类型

```
┌──────────────────────────────────────────────┐
│                                              │
│           【透视视口】Perspectives           │
│                                              │
│     类似人眼，近大远小，3D感最强              │
│     地编主要用这个！                         │
│                                              │
│              ↓ 点击切换 ↓                    │
│                                              │
├──────────────┬───────────────────────────────┤
│ 【顶视图】    │  【前视图】                   │
│ Top          │  Front                        │
│ (从上往下看)  │  (从正面看)                   │
│              │                               │
├──────────────┼───────────────────────────────┤
│ 【左视图】    │  【右视图】                   │
│ Left         │  Right                        │
│ (从左边看)    │  (从右边看)                   │
│              │                               │
└──────────────┴───────────────────────────────┘
```

💡 **切换方法**：视口左上角点击"透视"下拉菜单

## 二、导航操作（如何在场景中飞行）

### 2.1 三种导航方式对比

```
┌──────────────────────────────────────────────────────────────┐
│                     视口导航方式                              │
├──────────────┬──────────────────┬────────────────────────────┤
│   方式        │    操作           │        适合场景            │
├──────────────┼──────────────────┼────────────────────────────┤
│  鼠标导航     │ 按住右键         │ 最常用！地编日常操作        │
│              │ WASD移动         │                            │
├──────────────┼──────────────────┼────────────────────────────┤
│  Maya风格    │ Alt+中键         │ 有Maya基础的用户           │
│              │ 旋转平移缩放      │                            │
├──────────────┼──────────────────┼────────────────────────────┤
│  快捷键导航   │ 按住左键         │ 精确控制                   │
│              │ 配合快捷键        │                            │
└──────────────┴──────────────────┴────────────────────────────┘
```

### 2.2 鼠标导航（⭐重点掌握）

这是最常用的方式，记住这套操作你就自由了：

#### 基础移动（WASD）

```
                【鼠标右键按住 + WASD】
                        │
         ┌──────────────┼──────────────┐
         │              │              │
         ▼              ▼              ▼
        【W】          【S】          【A】
        向前          向后          向左
        (前进)       (后退)        (平移)
         │              │              │
         │              │              │
         │      ┌───────┴───────┐      │
         │      │               │      │
         └──────┤     【D】      ├──────┘
                │     向右      │
                │    (平移)     │
                └───────────────┘
```

**练习场景**：
```
想象你在玩第一人称游戏：
- 按住鼠标右键 = 进入"观察模式"
- W = 往前走
- S = 往后退
- A = 往左移
- D = 往右移

⚡ 鼠标往哪看，W就往哪走！
```

#### 高级控制

| 操作 | 效果 | 生活比喻 |
|------|------|----------|
| **鼠标右键 + 移动鼠标** | 环顾四周 | 就像你转头看四周 |
| **鼠标右键 + W + Shift** | 快速前进 | 从走路变跑步 |
| **鼠标右键 + W + 滚轮** | 调整移动速度 | 换挡变速 |
| **鼠标右键 + Q** | 下降/向下 | 直升机下降 |
| **鼠标右键 + E** | 上升/向上 | 直升机上升 |

#### 快速记忆口诀

```
🎮 "右键按住到处飞，WASD控制方向
     Q下降E上升，Shift加速不费劲儿"
```

### 2.3 缩放与聚焦

#### 缩放场景（不是缩放物体！）

```
【滚轮缩放】
   滚轮向前 = 拉近（放大）
   滚轮向后 = 拉远（缩小）
```

#### 聚焦选中物体（⭐超常用）

```
步骤：
1. 在大纲或场景中点击选中物体
2. 按 【F】 键
3. 视口自动飞到该物体面前

用途：场景大了找不到东西时，选中+F立刻找到！
```

#### 框选聚焦

```
按 【Shift + B】 → 框选区域 → 自动聚焦到该区域
```

## 四、变换操作（移动、旋转、缩放）

### 4.1 变换工具总览

```
                    物体变换三件套
                         │
        ┌────────────────┼────────────────┐
        │                │                │
        ▼                ▼                ▼
   ┌─────────┐     ┌─────────┐     ┌─────────┐
   │   移动  │     │   旋转  │     │   缩放  │
   │   (W)   │     │   (E)   │     │   (R)   │
   └────┬────┘     └────┬────┘     └────┬────┘
        │                │                │
        ▼                ▼                ▼
      [↔↕]            [⟲]            [⤢]
    红绿蓝轴        圆圈+切线       方块拉伸
    XYZ方向        旋转角度        比例大小
```

### 4.2 移动工具（W键）

```
选中物体后按 W，出现移动控件：

               Y轴（绿色）
                 ↑
                 │
                 │
    X轴（红色）←─┼─→
                 │
                 ↓
               Z轴（蓝色，向上/下）

操作：
• 拖动箭头 = 沿单一轴移动
• 拖动方块 = 沿平面移动（如XY平面）
• 底部小方块 = 自由移动
```

### 4.3 旋转工具（E键）

```
按 E 切换到旋转控件：

        红色圈 = 绕X轴旋转（像做前滚翻）
        绿色圈 = 绕Y轴旋转（像做侧手翻）
        蓝色圈 = 绕Z轴旋转（像转方向盘）
        
        白色外圈 = 自由旋转（不推荐，容易晕）

💡 按住Shift可以吸附角度（如15°增量）
```

### 4.4 缩放工具（R键）

```
按 R 切换到缩放控件：

        红色轴 = 只拉长/压缩 X 方向
        绿色轴 = 只拉长/压缩 Y 方向
        蓝色轴 = 只拉长/压缩 Z 方向
        中心白色方块 = 整体等比例缩放

⚠️ 注意：缩放不等于放大！缩放是改变物体的"比例"
```

### 4.5 世界坐标 vs 本地坐标

```
┌─────────────────────────────────────────────────────────────┐
│                  坐标系切换（重要！）                          │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   【世界坐标】World                                        │
│   无论物体怎么旋转，轴始终指向世界的XYZ                      │
│   就像地图上的东南西北，不会变                              │
│                                                             │
│      北(Y)                                                  │
│       ↑                                                     │
│  西 ──┼──→ 东(X)                                            │
│(X-)   │    (X+)                                             │
│       ↓                                                     │
│      南(Y-)                                                 │
│                                                             │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   【本地坐标】Local                                        │
│   轴跟随物体旋转而旋转                                      │
│   就像你手里的手机，转动时它的"上下"对你来说是固定的         │
│                                                             │
│   物体旋转45°后：                                           │
│        原Y轴 ↗                                              │
│              ╲                                              │
│               ╲ ← 现在的"前"                               │
│                ╲                                            │
│                 ╲                                           │
│   原X轴 ─────────→                                          │
│                                                             │
└─────────────────────────────────────────────────────────────┘

切换快捷键：Alt + ~（Tab上面那个键）
```

### 4.6 精确变换（细节面板）

```
选中物体 → 看右侧【细节面板】→ 【Transform】

┌──────────────────────────┐
│ Location 位置             │
│   X: [ 100.0 ]           │
│   Y: [ 200.0 ]           │
│   Z: [ 50.0  ]           │
├──────────────────────────┤
│ Rotation 旋转             │
│   X: [ 0.0   ] (Pitch俯仰)│
│   Y: [ 0.0   ] (Yaw偏航) │
│   Z: [ 45.0  ] (Roll翻滚)│
├──────────────────────────┤
│ Scale 缩放                │
│   X: [ 1.0   ]           │
│   Y: [ 1.0   ]           │
│   Z: [ 2.0   ] ← 拉高2倍 │
└──────────────────────────┘

💡 点击数值可以直接输入，按Tab跳到下一个
```

## 五、视图模式切换

### 5.1 常用视图模式

```
快捷键：Alt + 数字键

Alt + 1 = 线框模式      ← 看布线结构
Alt + 2 = 无光照模式    ← 看纯模型，不受光照影响  
Alt + 3 = 光照模式      ← ⭐最常用，正常预览
Alt + 4 = 细节光照      ← 看光照细节
Alt + 5 = 仅光照        ← 去掉贴图看光照
Alt + 6 = 复杂度        ← 看场景复杂度（优化用）
Alt + 7 = 光照重叠      ← 看光照优化（地编进阶）
```

### 5.2 游戏视图

```
快捷键：Alt + G

效果：隐藏所有编辑器辅助元素
  - 隐藏选择框
  - 隐藏坐标轴
  - 隐藏网格线
  
用途：预览最终游戏效果
```

## 六、实用技巧

### 6.1 视口相机书签

```
保存当前视角：Ctrl + 数字键(1-0)
读取保存视角：数字键(1-0)

场景：
  调整好一个好看的展示角度
  按 Ctrl + 1 保存
  以后按 1 就能立刻切回这个角度
```

### 6.2 最大化视口

```
快捷键：F11

效果：让当前视口占满整个屏幕
再次按F11恢复

用途：沉浸式工作，或截图展示
```

### 6.3 双屏工作

```
窗口 → 视口 → 添加视口

可以一边透视图，一边顶视图
或多显示器时拖出独立窗口
```
