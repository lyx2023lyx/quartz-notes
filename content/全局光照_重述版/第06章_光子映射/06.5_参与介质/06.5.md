# 6.5 参与介质

## 核心总结
参与介质是光线能在内部传播并散射的物质（如烟、雾、云），需要用体积渲染技术处理光的吸收、发射和散射。

---

## 什么是参与介质？

### 定义

**参与介质（Participating Media）**：
- 光可以在其中传播的物质
- 光会被吸收、散射
- 本身也可能发光

**例子**：
- 烟雾、云、雾
- 水、牛奶、果汁
- 皮肤、蜡、玉石
- 大气、灰尘

### 与表面的区别

**表面**：
- 光只在边界交互
- 要么反射，要么折射进入内部

**参与介质**：
- 光在内部传播时持续交互
- 逐渐衰减或散射

---

## 光在介质中的行为

### 吸收（Absorption）

**是什么**：
- 光能量被介质吸收
- 转化为其他形式（热）
- 光逐渐变弱

**吸收系数 σ_a**：
- 单位：1/长度
- 越大吸收越强
- 和波长有关（颜色选择性吸收）

**比尔-朗伯定律**：
```
T = exp(-σ_a × d)
```
- T：透射率
- d：传播距离

### 散射（Scattering）

**是什么**：
- 光子改变方向
- 但不是被吸收
- 继续传播

**散射系数 σ_s**：
- 单位：1/长度
- 越大散射越强

**散射类型**：

#### 瑞利散射（Rayleigh）
- 粒子比波长小
- 蓝光散射强（天空是蓝色的原因）
- 各向异性（前向和后向不同）

#### 米氏散射（Mie）
- 粒子比波长大
- 各向异性更强
- 例如云、雾

#### 各向同性散射
- 向各个方向均匀散射
- 简化模型

### 消光（Extinction）

**总衰减**：
```
σ_t = σ_a + σ_s
```
- σ_t：消光系数
- 吸收 + 散射

**平均自由程**：
```
l = 1/σ_t
```
- 光子平均传播多远才发生一次交互

## 体积渲染方程

### 与表面渲染方程的区别

**表面**：
- 只在边界积分
- 半球积分

**体积**：
- 沿光线积分
- 考虑内部每个点

### 体积渲染方程

```
L(x, ω) = ∫[0→d] T(x,y) × [σ_s(y) × L_s(y,ω) + L_e(y,ω)] dy
```

**各项含义**：
- x：观察点
- y：光线上的点
- d：到表面的距离
- T(x,y)：从x到y的透射率
- σ_s：散射系数
- L_s：散射光
- L_e：自发光

### 相位函数（Phase Function）

**是什么**：
- 描述散射方向分布
- 类似表面的BRDF

**各向同性**：
```
p(ω, ω') = 1/(4π)
```

**亨尼-格林斯坦（Henyey-Greenstein）**：
```
p(cosθ) = (1-g²) / (4π(1+g²-2g·cosθ)^(3/2))
```
- g：各向异性参数
- g=0：各向同性
- g>0：前向散射
- g<0：后向散射

## 渲染方法

### 1. 光线步进（Ray Marching）

**怎么做**：
1. 沿光线分成小步长
2. 每步采样介质属性
3. 累加贡献

**公式**：
```
对于步长Δt：
    透射 *= exp(-σ_t × Δt)
    散射 += 透射 × σ_s × L_s × Δt
```

**问题**：
- 步长大会漏细节
- 步长小计算慢

### 2. 随机游走（Random Walk）

**怎么做**：
- 像路径追踪一样
- 在介质内部随机采样交互点
- 散射后继续追踪

**步骤**：
1. 按指数分布采样距离
2. 移动到该点
3. 采样散射方向
4. 继续追踪

**终止**：
- 俄罗斯轮盘赌
- 或离开介质

### 3. 光子映射在介质中

**体积光子映射**：
- 光子在体积内部存储
- 3D空间密度估计
- 类似表面光子映射

**区别**：
- 3D查询（不是2D表面）
- 球形邻域（不是圆形）
- 更多光子需要

---

## 简化模型

### 单散射

**假设**：
- 只考虑一次散射
- 忽略多次散射

**适用**：
- 薄介质
- 烟雾、薄云
- 快速近似

### 多重散射

**是什么**：
- 光散射很多次
- 需要模拟或预计算

**方法**：
- 随机游走
- 扩散近似
- 球谐函数

---

## 实际应用

### 实时渲染

**简化技巧**：
- 体积纹理存储密度
- 低分辨率计算
- 时间累积降噪

**例子**：
- 游戏中的雾效
- 体积光（God Ray）
- 烟雾粒子

### 电影渲染

**高质量方法**：
- 随机游走
- 体积光子映射
- 多散射模拟

**例子**：
- 云（《阿凡达》《功夫熊猫》）
- 烟雾爆炸
- 水下场景

---

## 本节梳理

| 概念 | 含义 |
|------|------|
| 吸收 | 光能被介质吸收 |
| 散射 | 光子改变方向 |
| 消光 | 吸收+散射总衰减 |
| 平均自由程 | 平均传播距离 |
| 相位函数 | 散射方向分布 |

**渲染方法**：
- 光线步进：简单，适合均匀介质
- 随机游走：精确，适合复杂介质
- 体积光子映射：高质量，离线渲染

**关键公式**：
```
透射率 = exp(-消光系数 × 距离)
平均自由程 = 1/消光系数
```
