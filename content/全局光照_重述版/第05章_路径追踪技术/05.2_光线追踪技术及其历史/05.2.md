# 5.2 光线追踪技术及其历史

## 核心总结
光线追踪就是从相机往场景发射光线，计算光线和物体的交点，模拟光的真实传播路径。从1968年概念提出到现在，经历了从简单到复杂、从离线到实时的演变过程。

---

## 光线追踪是什么？

### 基本思想

**反过来想**：
- 真实世界：光线从光源发出 → 照到物体 → 反射 → 进入眼睛
- 光线追踪：从眼睛/相机发出光线 → 找到交点 → 计算颜色

**为什么反过来？**
- 光源发出的光大部分都浪费掉了（没进相机）
- 只计算"能看到的"光线，效率更高

### 基本流程

```
对于屏幕上的每个像素：
    1. 从相机发射一条光线穿过该像素
    2. 找到光线和场景中第一个物体的交点
    3. 计算该点的颜色
    4. 写入像素
```

## 光线追踪的发展历史

### 1968年：概念诞生

**Arthur Appel** 提出"ray casting"概念
- 从相机发射光线
- 找到最近交点
- 计算直接光照

**局限**：
- 只算直接光照
- 没有反射、折射、阴影

---

### 1979年：递归光线追踪

**Turner Whitted** 的经典论文
- 光线击中表面后产生新光线：
  - **反射光线**：镜面反射方向
  - **折射光线**：穿过透明物体
  - **阴影光线**：检测是否被遮挡

**效果**：
- 能算镜面反射（镜子、光滑金属）
- 能算折射（玻璃、水）
- 能算硬阴影

**局限**：
- 只算完美镜面/折射
- 漫反射表面看起来是"死黑"的
- 没有间接光照

---

### 1986年：渲染方程

**Jim Kajiya** 提出渲染方程
- 统一的光照计算框架
- 包含直接光照和间接光照
- 包含所有材质类型

**数学形式**：
```
L_o = L_e + ∫ f_r * L_i * cosθ dω
```

**意义**：
- 给光线追踪提供了理论基础
- 但直接求解太难（无穷维积分）

---

### 1990s：蒙特卡洛光线追踪

**突破**：用随机采样来解渲染方程

**路径追踪（Path Tracing）**：
- 从相机发射光线
- 击中表面后，随机选择反射方向
- 继续追踪，直到击中光源或跑太远
- 重复很多次，取平均

**优点**：
- 能算间接光照
- 能算软阴影
- 能算粗糙表面的反射

**缺点**：
- 噪点严重
- 需要大量采样
- 焦散效果难算

---

### 2000s：各种改进算法

#### 双向路径追踪（BDPT）
- 光线从相机和光源同时发射
- 在中间某处连接
- 提高复杂光照的效率

#### 光子映射（Photon Mapping）
- 先从光源发射"光子"
- 存储在光子图中
- 渲染时查询附近光子
- 专门擅长焦散效果

#### 梅特波利斯光照传输（MLT）
- 用马尔可夫链采样
- 擅长处理复杂可见性
- 适合处理困难光照场景

---

### 2010s-现在：实时光线追踪

**硬件突破**：
- NVIDIA RTX系列GPU（2018）
- 专用RT Core加速光线-三角形相交
- DirectX Raytracing (DXR) API

**技术特点**：
- 每像素只发射少量光线（1-4条）
- 配合降噪算法（Denoising）
- 混合渲染：光栅化+光线追踪

**应用场景**：
- 实时反射
- 实时阴影
- 实时全局光照

---

## 光线追踪 vs 光栅化

| 特性 | 光栅化 | 光线追踪 |
|------|--------|---------|
| 速度 | 快（硬件优化好） | 慢（计算量大） |
| 反射/折射 | 难做（要预计算） | 自然支持 |
| 阴影 | 用阴影贴图近似 | 精确计算 |
| 间接光照 | 难做 | 可以支持 |
| 场景复杂度 | 与三角形数量相关 | 可用加速结构优化 |

**当前趋势**：
- 实时渲染用光栅化+光线追踪混合
- 离线渲染用光线追踪
- 随着硬件发展，光线追踪会越来越普及

---

## 本节梳理

| 年代 | 里程碑 | 贡献 |
|------|--------|------|
| 1968 | Ray Casting | 基本光线投射 |
| 1979 | Whitted光线追踪 | 反射、折射、阴影 |
| 1986 | 渲染方程 | 统一理论框架 |
| 1990s | 蒙特卡洛方法 | 间接光照 |
| 2000s | BDPT/光子映射/MLT | 各种优化算法 |
| 2010s+ | GPU实时光追 | 走进游戏和实时应用 |

**核心演进**：
- 简单 → 复杂
- 近似 → 精确
- 离线 → 实时
