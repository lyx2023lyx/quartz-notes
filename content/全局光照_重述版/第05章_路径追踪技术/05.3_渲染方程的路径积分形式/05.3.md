# 5.3 渲染方程的路径积分形式

## 核心总结
渲染方程描述光怎么传播，路径积分形式把它改写成"从相机到光源的各种路径的贡献总和"，这样更容易用蒙特卡洛方法求解。

---

## 渲染方程回顾

### 原始形式
```
L_o(x, ω_o) = L_e(x, ω_o) + ∫ f_r(x, ω_i, ω_o) * L_i(x, ω_i) * cosθ dω_i
```

**各项含义**：
- L_o：从x点往ω_o方向发出的光
- L_e：x点自己发光的部分
- f_r：BRDF（材质反射特性）
- L_i：从ω_i方向来的入射光
- cosθ：入射角的影响
- dω_i：立体角微元

### 问题在哪？

**递归问题**：
- L_o 依赖于 L_i
- L_i 又是别的点的 L_o
- 无穷递归下去

**积分维度**：
- 半球积分（2D）
- 每个点都要做这个积分
- 直接求解几乎不可能

## 路径积分的思想

### 换个角度看问题

**传统思维**：
- 从相机出发
- 每次遇到表面都要积分
- 无穷递归

**路径积分思维**：
- 光从相机到光源走了一条具体的路径
- 计算这条路径的贡献
- 把所有可能路径的贡献加起来

### 什么是"路径"？

**定义**：
路径 = 相机 → 点1 → 点2 → ... → 点n → 光源

**路径上的点**：
- x_0：相机位置
- x_1, x_2, ..., x_k：光线经过的表面点
- x_{k+1}：光源

**路径长度**：
- 路径上的线段数（或点数）
- 例如：相机→墙面→灯泡，是2条线段（3个点）

---

## 路径积分的数学形式

### 简化版理解

**核心思想**：
```
最终图像 = 所有可能路径的贡献之和
```

**一条路径的贡献**：
```
P(路径) = 几何项 × BRDF乘积 × 光源亮度
```

**几何项**：
- 路径上每段距离的衰减
- 每对点之间的可见性
- 角度因子（cosθ）

**BRDF乘积**：
- 路径上每个反射点的BRDF值相乘
- 比如：镜面反射点用镜面BRDF，漫反射点用漫反射BRDF

### 具体公式

**路径贡献**：
```
f(x_0, x_1, ..., x_k) = 
    L_e(x_k → x_{k-1}) ×                    // 光源亮度
    ∏(i=1 to k-1) [f_r(x_{i+1}→x_i→x_{i-1}) × G(x_i ↔ x_{i+1})]
```

**几何项G**：
```
G(x ↔ y) = (cosθ_x × cosθ_y) / ||x-y||² × V(x,y)
```
- cosθ_x：x点法线和指向y的方向的夹角余弦
- ||x-y||²：两点距离的平方（衰减）
- V(x,y)：可见性（中间有没有遮挡）

## 路径分类

### 按路径长度分类

| 路径长度 | 描述 | 例子 |
|---------|------|------|
| L(E) | 直接看光源 | 灯泡直接进眼睛 |
| L(D)E | 光源→漫反射→眼睛 | 白墙反射灯光 |
| L(S)E | 光源→镜面→眼睛 | 镜子反射灯泡 |
| L(D,D)E | 光源→漫反射→漫反射→眼睛 | 光墙→白墙→眼睛 |
| L(S,D)E | 光源→镜面→漫反射→眼睛 | 灯泡→镜子→白墙→眼睛 |

**符号说明**：
- L：光源（Light）
- E：眼睛/相机（Eye）
- D：漫反射（Diffuse）
- S：镜面/光泽（Specular）

### 为什么分类？

**不同路径计算难度不同**：
- L(D)E：简单，直接算
- L(S,S,S...)E：困难，镜面反射方向单一
- L(D,D,D...)E：中等，需要多次积分

**实际应用**：
- 有些算法专门处理特定类型路径
- 比如光子映射擅长处理L(S,D)E（焦散）

---

## 路径积分的好处

### 1. 适合蒙特卡洛

**为什么**：
- 路径积分把渲染方程变成"求和"形式
- 蒙特卡洛擅长估计求和
- 随机采样一些路径，算它们的贡献，取平均

### 2. 灵活选择采样策略

**可以**：
- 从相机出发采样（路径追踪）
- 从光源出发采样（光路追踪）
- 两边同时出发，中间连接（双向路径追踪）

### 3. 容易分析算法特性

**可以分析**：
- 哪些路径算得好
- 哪些路径算得差
- 怎么改进采样

---

## 实际计算中的问题

### 路径空间太大

**问题**：
- 可能的路径无穷多
- 每条路径还有无穷多变体（点可以无限细分）

**解决**：
- 只采样一小部分路径
- 用概率密度函数（PDF）指导采样
- 重要性采样：多采重要的路径

### 路径贡献差异大

**问题**：
- 有些路径贡献很大（直接照到光源）
- 有些路径贡献很小（经过多次反射几乎没光）

**解决**：
- 俄罗斯轮盘赌（Russian Roulette）：
  - 随机决定是否继续追踪
  - 贡献小的路径大概率终止
  - 省下的计算资源用于重要路径

## 本节梳理

| 概念 | 简单理解 |
|------|---------|
| 路径 | 相机→点1→点2→...→光源 |
| 路径贡献 | 这条路径带来的亮度 |
| 几何项 | 距离衰减+角度+可见性 |
| BRDF乘积 | 每个反射点的材质特性 |
| 路径分类 | L(D)E, L(S)E等，不同难度 |
| 蒙特卡洛 | 随机采一些路径估算总和 |

**核心公式记忆**：
```
最终图像 = ∑(所有路径的贡献)
路径贡献 = 光源亮度 × BRDF乘积 × 几何项
```

**关键理解**：
- 路径积分把递归问题变成求和问题
- 适合用蒙特卡洛随机采样
- 不同路径用不同策略采样效率更高
