# 8.5 Enlighten中间件

## 核心总结
Enlighten是业界广泛使用的实时辐射度解决方案，预计算辐射度传输，支持动态光源和材质变化。

---

## 什么是Enlighten？

### 商业中间件

**开发者**：
-  originally by Geomerics
-  现在被Unity收购

**应用**：
-  游戏（Battlefield, FIFA等）
-  实时应用
-  建筑可视化

### 核心能力

**实时全局光照**：
-  静态几何
-  动态光源
-  动态材质

**特点**：
-  预计算 + 实时求解
-  适合游戏场景

---

## 技术原理

### 预计算阶段

**辐射度传输**：
-  计算形状系数
-  存储传输关系
-  压缩数据

**聚类**：
-  把相似的片聚类
-  减少计算量
-  层次结构

### 实时阶段

**求解**：
-  光源变化 → 重新求解
-  材质变化 → 重新求解
-  但传输关系不变

**优化**：
-  只更新变化的部分
-  渐进式更新
-  多线程并行

---

## 工作流程

### 烘焙（Bake）

```
1. 导入场景
2. 标记静态物体
3. 设置细分参数
4. 计算辐射度传输
5. 导出数据
```

### 运行时

```
1. 加载预计算数据
2. 设置光源
3. 实时求解辐射度
4. 应用到渲染
```

---

## 优缺点

### 优点

-  实时性能好
-  支持动态光源
-  集成方便
-  质量高

### 缺点

-  只能静态几何
-  预计算时间长
-  内存占用大
-  商业授权

---

## 本节梳理

**Enlighten = 预计算辐射度 + 实时求解**

**适用**：
-  室内游戏场景
-  建筑可视化
-  需要动态光照

**限制**：
-  静态几何
-  商业软件
