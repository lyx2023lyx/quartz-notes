# 8.3 求解线性方程组

## 核心总结
辐射度方法最终归结为求解线性方程组，可以用直接法或迭代法，迭代法更适合大规模场景。

---

## 辐射度方程

### 矩阵形式

```
B = E + ρFB
```

**变形**：
```
(I - ρF)B = E
```

**求解**：
```
B = (I - ρF)^(-1) × E
```

**各项**：
- B：亮度向量（待求）
- E：自发光向量
- ρ：反射率对角矩阵
- F：形状系数矩阵
- I：单位矩阵

---

## 求解方法

### 1. 直接法

**高斯消元**：
- 把矩阵变成上三角
- 回代求解

**问题**：
- 复杂度O(N³)
- N=1000时太慢
- 不实用

### 2. 迭代法（常用）

#### 雅可比迭代

**公式**：
```
B_i^(k+1) = E_i + ρ_i × Σ(F_ij × B_j^(k))
```

**步骤**：
1. 初始化B^(0) = E
2. 用旧值计算新值
3. 重复直到收敛

**特点**：
- 简单
- 收敛慢

#### 高斯-赛德尔迭代

**改进**：
- 用已经算出的新值
- 而不是全部用旧值

**公式**：
```
B_i^(k+1) = E_i + ρ_i × (Σ(j<i) F_ij × B_j^(k+1) + Σ(j>i) F_ij × B_j^(k))
```

**特点**：
- 收敛更快
- 实际常用

#### 逐步求精（Progressive Refinement）

**思想**：
- 一次只处理最亮的片
- 把它的光传到其他片
- 类似光线追踪

**步骤**：
1. 找当前最亮的片
2. 计算它对其他片的贡献
3. 更新其他片的亮度
4. 重复

**优点**：
- 可以随时停止（近似结果）
- 适合实时预览

---

## 收敛判断

### 停止条件

**误差检查**：
```
||B^(k+1) - B^(k)|| < ε
```

**或者**：
- 固定迭代次数
- 达到时间限制

### 加速收敛

**超松弛（SOR）**：
```
B_new = (1-ω)B_old + ωB_gauss_seidel
```

**ω > 1**：加速收敛
**ω < 2**：保证收敛

---

## 本节梳理

**求解方程**：
```
B = (I - ρF)^(-1) × E
```

**方法选择**：
- 直接法：太小不用，太大太慢
- 迭代法：实用选择
- 逐步求精：可中断，适合实时

**收敛加速**：
- 高斯-赛德尔比雅可比快
- SOR进一步加速
