# 8.2 形状系数

## 核心总结
形状系数描述两个曲面片之间传递光照的几何关系，只和位置、朝向、可见性有关，和亮度无关，可以预计算。

---

## 形状系数是什么？

### 直观理解

**定义**：
片A发出的光，有多大比例到达片B

**取值范围**：
- 0：完全看不到
- 1：全部到达（实际上不可能，因为还有其他方向）

**影响因素**：
1. 距离：越远越小
2. 角度：正对时最大，斜对时变小
3. 可见性：中间有遮挡为0

---

## 形状系数公式

### 基本公式

```
F_ij = (1/A_i) × ∫∫ (cosθ_i × cosθ_j) / (π × r²) × V_ij dA_i dA_j
```

**各项含义**：
- F_ij：从片i到片j的形状系数
- A_i：片i的面积
- θ_i, θ_j：两片面法线和连线夹角
- r：两片之间的距离
- V_ij：可见性（0或1）

### 几何意义

**两次投影**：
- cosθ_i：片i投影到传输方向
- cosθ_j：片j投影到传输方向

**距离衰减**：
- 1/r²：和距离平方成反比

**可见性**：
- V_ij = 1：能看到
- V_ij = 0：被遮挡

**对应图片讲解**：
- 图片展示了两片面之间的几何关系
- 展示了角度投影和距离

---

## 形状系数的性质

### 1. 互易性

```
A_i × F_ij = A_j × F_ji
```

**含义**：
小片向大片传的比例大，但乘上面积后相等

### 2. 封闭性

```
Σ F_ij = 1（对所有j求和）
```

**含义**：
一个片发出的光，必定传到某些地方（包括可能的环境）

### 3. 可加性

片可以分割，形状系数也分割

---

## 计算方法

### 1. 解析法

**简单形状**：
- 平行矩形
- 垂直矩形
- 有现成的公式

**局限**：
- 复杂形状没解析解
- 实际很少用

### 2. 蒙特卡洛积分

**半立方体法**：
1. 在片i上方放一个半立方体
2. 把半立方体面分成小格子
3. 投影其他片到半立方体
4. 计算覆盖的格子比例

**优点**：
- 容易实现
- 可以处理遮挡

### 3. 光线追踪法

**从片i随机发射光线**：
1. 在单位半球均匀采样方向
2. 追踪光线
3. 如果击中片j，计数
4. 比例 ≈ 形状系数

**公式**：
```
F_ij ≈ (击中j的光线数) / (总光线数)
```

**优点**：
- 精确
- 容易处理复杂几何

**缺点**：
- 慢
- 需要很多光线

---

## 预计算和存储

### 矩阵形式

**形状系数矩阵F**：
```
    j=1  j=2  j=3 ...
i=1 [F11  F12  F13 ...]
i=2 [F21  F22  F23 ...]
i=3 [F31  F32  F33 ...]
...
```

**大小**：
- N个片 → N×N矩阵
- N=1000 → 100万个元素
- 内存占用大

### 压缩存储

**稀疏性**：
- 远处的片F_ij≈0
- 只存非零元素

**层次结构**：
- 远处的片合并成大片
- 减少矩阵大小

---

## 本节梳理

**形状系数 = 几何传递因子**

**计算公式**：
```
F_ij ∝ (cosθ_i × cosθ_j) / r² × 可见性
```

**计算方法**：
- 解析法：简单形状
- 半立方体：经典方法
- 光线追踪：精确但慢

**存储**：
- N×N矩阵
- 通常需要压缩
