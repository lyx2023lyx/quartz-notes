# 7.1 马尔可夫链与梅特波利斯算法

## 核心总结
梅特波利斯算法用马尔可夫链在路径空间中随机游走，根据贡献大小决定是否接受新路径，从而集中采样重要路径。

---

## 为什么要用梅特波利斯算法？

### 路径追踪的困难

**困难光照场景**：
- 小窗户照进的室内光
- 镜子反射的间接光
- 复杂遮挡的场景

**问题**：
- 随机采样很难击中这些路径
- 大部分采样浪费
- 噪点集中在困难区域

### 打个比方

**找宝藏**：
- 路径追踪：随机挖，大部分地方没宝藏
- 梅特波利斯：找到一处宝藏后，在附近继续挖

---

## 马尔可夫链基础

### 什么是马尔可夫链？

**定义**：
- 一系列状态
- 下一个状态只依赖当前状态
- 和之前的历史无关

**例子**：
- 天气：今天晴天→明天可能晴天/雨天
- 只依赖今天，不依赖昨天

**路径空间中的应用**：
- 状态 = 一条光线路径
- 下一个路径 = 当前路径 + 小的修改

### 稳态分布

**概念**：
- 运行足够长时间后
- 访问各状态的概率趋于稳定

**重要性**：
- 梅特波利斯算法可以设计稳态分布
- 让重要的路径被访问更多次

---

## 梅特波利斯算法

### 核心思想

**目标**：
以概率正比于贡献的方式采样路径

**公式**：
```
访问概率 ∝ 路径贡献 f(x)
```

**步骤**：
1. 从当前路径 x 出发
2. 提出新路径 y（通过突变）
3. 计算接受概率
4. 决定是否接受 y

### 接受概率

**公式**：
```
P(接受) = min(1, f(y)/f(x) × T(x|y)/T(y|x))
```

**各项含义**：
- f(y), f(x)：新/旧路径的贡献
- T(y|x)：从x到y的转移概率
- T(x|y)：从y到x的转移概率

**简化（对称突变）**：
```
如果 T(y|x) = T(x|y)：
P(接受) = min(1, f(y)/f(x))
```

**直观理解**：
- 新路径更好（f(y) > f(x)）：一定接受
- 新路径更差：按概率接受
- 允许偶尔接受差路径，避免局部最优

## 突变策略

### 什么是突变？

**定义**：
对当前路径做小修改，产生新路径

**要求**：
- 容易计算
- 保持路径有效性
- 能探索路径空间

### 常见突变类型

#### 1. 双向突变（Bidirectional Mutation）

**做法**：
1. 随机选路径上的一个顶点
2. 删除该顶点之后的子路径
3. 从该顶点随机发射新子路径

**特点**：
- 改变路径的后半部分
- 保持前半部分

#### 2. lens突变

**做法**：
1. 改变相机光线在图像平面上的起点
2. 保持其余路径不变

**特点**：
- 在图像上移动
- 保持3D路径形状

#### 3. 多链突变（Multi-chain Mutation）

**做法**：
1. 选路径上的多个顶点
2. 分别修改这些顶点

**特点**：
- 改变多个部分
- 大步长探索

#### 4. 扰动突变（Perturbation）

**做法**：
- 小范围扰动方向
- 保持大致路径

**特点**：
- 小步长精细化
- 在好路径附近优化

## 算法流程

### 初始化

```
1. 用普通路径追踪生成初始路径
2. 计算初始路径贡献
3. 当前路径 = 初始路径
```

### 迭代

```
对于每次迭代：
    1. 对当前路径应用突变 → 新路径
    2. 计算新路径贡献
    3. 计算接受概率
    4. 生成随机数
    5. 如果随机数 < 接受概率：
          当前路径 = 新路径
    6. 记录当前路径贡献到对应像素
```

### 重要技巧：起始偏移

**问题**：
- 初始路径可能很差
- 需要burn-in时间

**解决**：
- 前N次迭代不记录
- 让马尔可夫链达到稳态

---

## 优缺点

### 优点

1. **擅长困难场景**
   - 小窗户、复杂遮挡
   - 一旦找到路径，会在附近探索

2. **样本复用**
   - 好路径附近产生更多样本
   - 自动集中于重要区域

3. **一致性**
   - 无限时间后收敛
   - 理论保证

### 缺点

1. **启动慢**
   - 需要burn-in
   - 初始样本质量不稳定

2. **序列相关**
   - 样本之间不独立
   - 估计方差难计算

3. **参数调优**
   - 突变策略选择
   - 突变大小调整

4. **简单场景慢**
   - 不如普通路径追踪直接
   - 复杂度过高

---

## 本节梳理

**核心思想**：
```
马尔可夫链游走 + 梅特波利斯接受准则 = 集中采样重要路径
```

**关键公式**：
```
接受概率 = min(1, 新贡献/旧贡献)
```

**适用场景**：
- 困难光照路径
- 复杂可见性
- 离线高质量渲染

**不适合**：
- 简单场景
- 实时渲染
