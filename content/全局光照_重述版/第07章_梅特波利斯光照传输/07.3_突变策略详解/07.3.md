# 7.3 突变策略详解

## 核心总结
突变策略决定如何修改当前路径产生新路径，不同策略各有优势，组合使用能更高效地探索路径空间。

---

## 双向突变（Bidirectional Mutation）

### 原理

**步骤**：
1. 随机选路径上的一个顶点（除端点外）
2. 删除该顶点之后的子路径
3. 从该顶点随机发射新子路径

**示意图**：
```
原路径：相机 → A → B → C → D → 光源
选顶点B
新路径：相机 → A → B → E → F → 光源
```

**优点**：
- 改变路径后半部分
- 保持前半部分（通常更重要）
- 容易实现

**缺点**：
- 可能产生无效路径（不可见）
- 需要检查可见性

---

## Lens突变

### 原理

**步骤**：
1. 改变相机光线在像平面上的起点
2. 保持3D路径形状大致不变

**几何解释**：
- 原路径穿过像素(i,j)
- 新路径穿过像素(i+Δi, j+Δj)
- 从新的像素位置重新投射光线

**特点**：
- 在图像空间移动
- 适合探索相邻像素的光照
- 接受率通常较高

---

## 扰动突变（Caustic Perturbation）

### 原理

**针对焦散路径**：
- 保持镜面反射/折射方向大致不变
- 小范围扰动方向

**步骤**：
1. 识别路径中的镜面顶点
2. 对这些顶点的出射方向做小扰动
3. 重新计算后续路径

**适用场景**：
- 焦散效果（Pool caustics）
- 镜面反射链

**扰动大小**：
- 通常用高斯分布
- 标准差控制扰动幅度

---

## 多链突变（Multi-chain Mutation）

### 原理

**同时修改多个顶点**：
1. 选多个顶点（通常2-3个）
2. 分别生成新子路径连接它们

**优点**：
- 大步长探索
- 可能找到完全不同的路径

**缺点**：
- 接受率低
- 计算量大

**使用频率**：
- 占突变总数的10-20%
- 防止陷入局部最优

---

## 策略选择

### 根据场景选择

| 场景特征 | 推荐突变策略 |
|---------|-------------|
| 直接光照为主 | Lens突变 |
| 复杂间接光照 | 双向突变 |
| 焦散效果 | 扰动突变 |
| 困难场景 | 多链突变 |

### 自适应策略

**动态调整**：
- 监控各突变的接受率
- 接受率高的多使用
- 接受率低的少使用

**实现**：
```
每N次迭代统计一次：
- 突变A接受率30% → 增加权重
- 突变B接受率5% → 减少权重
```

---

## 本节梳理

**突变类型对比**：

| 突变类型 | 改变范围 | 接受率 | 适用场景 |
|---------|---------|--------|---------|
| 双向突变 | 后半路径 | 中等 | 通用 |
| Lens突变 | 像平面位置 | 高 | 探索图像空间 |
| 扰动突变 | 方向微调 | 高 | 焦散、镜面 |
| 多链突变 | 多个顶点 | 低 | 大步长探索 |

**组合策略**：
- 70% 小步长（双向、扰动）
- 30% 大步长（多链、随机）
