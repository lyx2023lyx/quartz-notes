# 1.2 辐射度量学

## 核心总结
辐射度量学就是一套测量"光"的方法，定义了几个基本物理量（能量、功率、亮度等），让我们能精确计算光的传播和物体表面的光照。

---

## 为什么要学这个？

### 打个比方
想象你要做一道菜：
- 没有度量衡："加一点盐"、"炒一会儿"——做出来味道不稳定
- 有了度量衡："加5克盐"、"炒3分钟"——每次都能复现

**辐射度量学就是给"光"定义的度量衡**，让我们能精确计算：
- 光源发出了多少光？
- 物体表面接收了多少光？
- 眼睛/相机看到了多少光？

---

## 前置知识：立体角

### 什么是立体角？

**平面角**：在2D平面上，两条射线张开的角度（用度或弧度表示）

**立体角**：在3D空间中，一个锥形"张开"的大小

### 怎么理解立体角？

想象你站在球心：
- 平面角：看地面上的一块区域，只能左右看
- 立体角：可以看天空中的任意方向，是一个"锥形"的范围

### 单位：球面度（sr）

- 整个球面的立体角 = 4π sr（约12.57）
- 半球面的立体角 = 2π sr
- 正方体一个面从中心看的立体角 = π/3 sr

**对应图片讲解**：
- 图片展示了立体角的定义：单位球面上某块区域的面积就是这块区域对应的立体角大小

## 五个基本物理量

### 1. 辐射能量（Radiant Energy）

**符号**：Q  
**单位**：焦耳（J）

**是什么**：
光的总能量。就像你说"这个灯泡消耗了多少电"，指的是一段时间内的总能量。

**类比**：
就像你跑步消耗的热量，跑10分钟消耗了500焦耳——这是总能量。

**公式**：
```
Q = hν
```
- h：普朗克常数（一个很小的数）
- ν：光的频率（蓝光频率高，红光频率低）

**简单理解**：
- 蓝光光子能量高（频率高）
- 红光光子能量低（频率低）
- 这就是为什么蓝紫光对皮肤伤害更大

---

### 2. 辐射通量（Radiant Flux）

**符号**：Φ（大写phi）  
**单位**：瓦特（W）

**是什么**：
光源每秒发出的功率。就像你说"这个灯泡是100瓦的"，指的是它每秒钟发出100焦耳的光能量。

**公式**：
```
Φ = dQ/dt
```
就是能量对时间求导（能量随时间的变化率）。

**类比**：
- 能量Q = 你钱包里的总钱数
- 通量Φ = 你每秒钟赚的钱（或者花的钱）

**为什么重要**：
在图形学里，我们关心的是"每秒钟"的光照，而不是"总共"有多少光。所以我们主要用这个量。

---

### 3. 辐射亮度（Radiance）

**符号**：L  
**单位**：瓦特/(平方米·球面度) [W/(m²·sr)]

**是什么**：
这是最重要的一个量！表示"一束光"有多亮。

**三个要素**：
1. 从哪个点发出的？
2. 往哪个方向发出的？
3. 有多亮？

**为什么最重要？**
- 人眼感知的就是辐射亮度
- 相机记录的也是辐射亮度
- 渲染的最终目标就是算出每个像素点的辐射亮度

**公式**：
```
dΦ = L·cosθ·dA·dω
```

**拆解**：
- dΦ：这束光的功率
- L：亮度（我们要算的量）
- cosθ：表面倾斜角度的影响（斜着看就没那么亮）
- dA：这束光来自多大的面积
- dω：这束光占多大的立体角

**类比**：
想象用手电筒照墙：
- 正对着照：一个很亮的小圆斑
- 斜着照：一个较暗的大椭圆斑
- 这就是cosθ的影响

**重要特性**：
**辐射亮度不随距离变化！**

- 听起来很反直觉对吧？
- 实际上，虽然光确实会随距离变暗，但那是因为光线发散覆盖的面积变大了
- 如果你只看"一束光"本身，它的亮度是不变的
- 这就是为什么我们能在地球上看到太阳，虽然它离我们很远

**对应图片讲解**：
- 图片展示了辐射亮度的定义：某个点往某个方向的亮度

---

### 4. 辐射强度（Radiant Intensity）

**符号**：I  
**单位**：瓦特/球面度 [W/sr]

**是什么**：
光源往某个方向发出的"总亮度"，不考虑距离。

**公式**：
```
I = ∫ L·cosθ·dA
```
就是把整个发光面上所有点的亮度加起来。

**什么时候用**：
- 描述光源本身的特性
- 比如一个聚光灯：中间最亮，边缘逐渐变暗
- 这种"不同方向不同亮度"的特性用辐射强度来描述

**朗伯余弦定理**：
对于理想的漫反射表面（比如完美的白墙）：
```
I(θ) = I₀·cosθ
```
意思是：斜着看的时候，亮度按cosθ衰减。

**为什么？**
- 正对着看：能看到完整的表面
- 斜着看：只能看到"投影"后的表面，面积变小了
- 所以看起来变暗了

**对应图片讲解**：
- 图片展示了灯泡在不同方向可能有不同的辐射强度
- 另一张图展示了漫反射表面的余弦衰减

### 5. 辐射照度（Irradiance）

**符号**：E  
**单位**：瓦特/平方米 [W/m²]

**是什么**：
物体表面接收到的光的总功率，除以表面积。

**简单说**：
就是"单位面积上接收到了多少光"。

**公式**：
```
E = ∫ L·cosθ·dω
```
把来自所有方向的光都积分起来。

**和辐射亮度的区别**：
- 辐射亮度L：一束光，有方向性
- 辐射照度E：一个点接收的所有光的总和，没有方向性

**类比**：
- L（亮度）：一束激光打在墙上，很亮但范围小
- E（照度）：整个房间开灯，每平米接收到的光

**平方反比定律**：
点光源产生的照度与距离的平方成反比：
```
E = I·cosθ / r²
```

**为什么？**
想象气球被吹大：
- 距离1米：光分布在面积为4πr²的球面上
- 距离2米：球面面积变成4倍(4π(2r)²=16πr²)
- 同样的光分散在4倍的面积上
- 所以每平米的光只有1/4

**实际应用**：
在图形学里，我们很少直接用平方反比，而是用"距离衰减函数"：
```
E = I·cosθ·f_dist(r)
```

**为什么要改？**
1. **控制性**：可以调参数达到艺术效果
2. **性能**：设置最大距离，超出范围就不计算了
3. **避免无穷大**：距离趋近0时，真实的平方反比会趋近无穷大，计算机处理不了

**对应图片讲解**：
- 图片展示了点光源产生的辐射照度
- 另一张图展示了平行光（太阳光）的照度不随距离变化

## 本节梳理

| 物理量 | 符号 | 单位 | 简单理解 |
|--------|------|------|---------|
| 辐射能量 | Q | 焦耳(J) | 光的总能量 |
| 辐射通量 | Φ | 瓦特(W) | 每秒发出的光 |
| 辐射亮度 | L | W/(m²·sr) | 一束光的亮度（最重要）|
| 辐射强度 | I | W/sr | 光源往某方向的总亮度 |
| 辐射照度 | E | W/m² | 表面接收的光 |

**关键公式记忆**：
1. **亮度定义**：dΦ = L·cosθ·dA·dω
2. **照度公式**：E = I·cosθ / r²
3. **朗伯定理**：I(θ) = I₀·cosθ

**最重要的概念**：
**辐射亮度L**是渲染的核心——我们要算的就是每个像素点、往相机方向的辐射亮度！
