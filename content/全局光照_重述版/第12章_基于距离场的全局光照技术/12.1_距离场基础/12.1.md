# 12.1 距离场基础

## 核心总结
距离场存储空间中每个点到最近表面的距离，是隐式表面表示，可用于快速光线追踪和可见性计算。

---

## 什么是距离场？

### 定义

**有符号距离场（SDF）**：
- 每个点存储到最近表面的距离
- 内部为负，外部为正
- 表面为0

### 例子

**圆**：
- 圆内：距离为负（到圆边界的距离）
- 圆外：距离为正
- 圆上：距离为0

### 存储方式

**3D纹理**：
- 均匀采样空间
- 每个体素存距离值

**自适应**：
- 八叉树存储
- 只在表面附近细分

---

## 距离场的性质

### 1. 梯度是法线

**公式**：
```
法线 = ∇SDF / |∇SDF|
```

**意义**：
- 距离变化最快的方向
- 就是表面法线方向

### 2. 安全步进

**性质**：
- 当前点的距离值 = 安全步进距离
- 保证不会穿过表面

**应用**：
- 球体追踪
- 加速光线遍历

### 3. 布尔操作

**并集**：
```
SDF_union = min(SDF_A, SDF_B)
```

**交集**：
```
SDF_intersection = max(SDF_A, SDF_B)
```

**差集**：
```
SDF_difference = max(SDF_A, -SDF_B)
```

---

## 生成方法

### 1. 解析法

**简单形状**：
- 球、盒、圆柱
- 有解析公式

### 2. 从网格生成

**方法**：
- 对每个体素，找最近三角形
- 计算距离
- 判断内外（法线方向）

**加速**：
- 空间加速结构
- 只检查附近的三角形

### 3. 扫描法

**2D距离变换**：
- 先水平扫描
- 再垂直扫描
- 快速计算

**扩展到3D**：
- 三个方向扫描

---

## 本节梳理

**距离场 = 到最近表面的有符号距离**

**性质**：
- 梯度是法线
- 安全步进
- 布尔操作简单

**生成**：
- 解析法
- 从网格生成
- 扫描法
