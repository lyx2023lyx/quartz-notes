# 🚀 Learn Claude Code - 从0到1手写AI编程助手

> [!info] 项目简介
> 这是一个从零开始构建类似 **Claude Code** 的AI编程助手的教学项目。通过学习本项目，你将深入理解AI Agent的核心架构和实现原理。

## 🎯 核心目标

学完本项目后，你将能够：

- [x] 理解AI Agent的基本架构和工作原理
- [x] 掌握与LLM（大语言模型）的交互方式
- [ ] 实现工具调用（Tool Calling）机制
- [ ] 构建对话上下文管理系统
- [ ] 集成文件系统操作能力
- [ ] 打造一个可用的AI编程助手
